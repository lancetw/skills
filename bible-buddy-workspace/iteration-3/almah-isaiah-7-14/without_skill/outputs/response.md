# 以賽亞書 7:14 中的 'almah' 到底是什麼意思？處女生子的預言是怎麼來的？

## 希伯來原文：'almah' 的語義範圍

以賽亞書 7:14 的希伯來原文使用的是 **עַלְמָה (almah)**，這個詞在希伯來聖經中出現了大約 7 次（創世記 24:43、出埃及記 2:8、詩篇 68:25、箴言 30:19、雅歌 1:3、6:8，以及以賽亞書 7:14）。

**'almah' 的核心含義是「年輕女子」（young woman）**，特別指已達到適婚年齡的少女。這個詞的語義重點在於「年輕」，而非特指生理上的處女身份。

希伯來文中另有一個更明確表示「處女」的詞：**בְּתוּלָה (betulah)**，這個詞在舊約中出現約 50 次，通常被理解為直接指涉未有性經驗的女子（雖然學者對 betulah 是否「每次」都嚴格指處女也有一些討論）。

如果以賽亞的本意是強調「處女」的概念，他更可能會使用 betulah 而非 almah。不過，也有保守派學者指出，almah 在舊約中所指的女性事實上都是未婚少女，因此在當時的文化脈絡下，almah 隱含著貞潔的意涵。

## 以賽亞書 7:14 的原始歷史脈絡

這段經文的歷史背景是**亞蘭-以法蓮戰爭**（約公元前 735 年）。當時亞蘭王利汛和北國以色列王比加聯手進攻南國猶大，猶大王亞哈斯非常恐懼。先知以賽亞奉命前去安慰亞哈斯，告訴他這個威脅不會持久：

> 「因此，主自己要給你們一個兆頭：看哪，必有一位 almah 懷孕生子，給他起名叫以馬內利。」（以賽亞書 7:14）

從上下文（7:15-16）來看，這個孩子的成長是一個時間標記——在這孩子還不懂分辨善惡之前，那兩個威脅猶大的王就會被除滅。這意味著在原始脈絡中，這個預言指向的是**當時即將發生的事件**，而非遙遠的未來。那位 almah 很可能是：

- 以賽亞的妻子（「女先知」，參見 8:3）
- 亞哈斯的妻子（即後來的希西家王之母）
- 或當時某位為人所知的年輕女子

## 七十士譯本（Septuagint）的關鍵轉譯

「處女生子」概念的形成，有一個非常關鍵的翻譯節點。

約在公元前 3-2 世紀，猶太學者在埃及亞歷山大城將希伯來聖經翻譯為希臘文，即**七十士譯本（Septuagint, LXX）**。在翻譯以賽亞書 7:14 時，他們將 almah 翻譯為希臘文 **παρθένος (parthenos)**，而 parthenos 在希臘文中的主要含義就是「處女」（virgin）。

這個翻譯選擇可能的原因包括：

1. **語義重疊**：在古代近東文化中，年輕未婚女子通常也被預設為處女，因此譯者可能認為 parthenos 是合理的對等翻譯
2. **神學強化**：譯者可能有意強調這個「兆頭」的超自然性質——既然是上帝親自給的兆頭，一般的懷孕似乎不夠特別
3. **文化語境**：在希臘文化中，parthenos 也不是每次都嚴格指生理意義上的處女，有時也泛指年輕女子

## 新約的引用與基督教詮釋

馬太福音 1:22-23 直接引用了以賽亞書 7:14 的七十士譯本版本：

> 「這一切的事成就，是要應驗主藉先知所說的話，說：『必有童女（parthenos）懷孕生子，人要稱他的名為以馬內利。』」

馬太的寫作目的是向猶太讀者論證耶穌就是舊約預言的彌賽亞。他使用的是當時猶太社群廣泛使用的希臘文聖經（LXX），並將這段經文理解為**預表論（typology）**——即舊約事件同時也是未來更大事件的「預影」。

在這個詮釋框架下：

- **近期應驗**：以賽亞時代的那位年輕女子生了一個兒子，作為神同在的記號
- **終極應驗**：馬利亞以童貞之身生下耶穌，完全實現了「以馬內利」（神與我們同在）的意義

## 各方立場摘要

| 立場 | 觀點 |
|------|------|
| **猶太傳統** | almah 指年輕女子，預言在以賽亞時代已應驗，與處女生子無關 |
| **基督教傳統** | 承認近期應驗，但認為終極應驗在耶穌身上；七十士譯本的 parthenos 翻譯受到聖靈引導 |
| **歷史批判學** | almah 不等於處女；馬太基於 LXX 的 parthenos 進行了神學詮釋 |
| **保守福音派** | almah 雖不嚴格等於處女，但在文化上隱含貞潔意涵；神的預言可以有雙重應驗 |

## 總結

'almah' 在希伯來文中的基本含義是「年輕女子」，而非嚴格意義上的「處女」。「處女生子」的預言概念主要經由以下路徑形成：

1. 希伯來文 almah（年輕女子）
2. 被七十士譯本翻譯為希臘文 parthenos（處女）
3. 馬太福音引用 LXX 版本，將其應用於馬利亞的童貞懷孕
4. 基督教神學傳統將此確立為核心教義

這是一個翻譯選擇、文化假設、與神學詮釋交織產生的結果。無論個人的信仰立場如何，理解這段歷史脈絡對於嚴肅地研讀聖經都是不可或缺的。
