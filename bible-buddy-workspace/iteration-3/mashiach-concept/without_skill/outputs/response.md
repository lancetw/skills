# 第一世紀猶太人的「彌賽亞」(משיח) 概念 vs. 今日教會的「基督」

## 一、詞源與基本意義

「彌賽亞」(משיח, Mashiach) 的希伯來文原意是「受膏者」——指被倒上聖膏油、被神揀選承擔特殊使命的人。在舊約中，這個詞並非專指某一位末世人物，而是泛指被膏立的祭司（利 4:3）、君王（撒上 24:6 稱掃羅為「耶和華的受膏者」）、甚至外邦君王（賽 45:1 稱波斯王居魯士為「我的受膏者」）。

到了第二聖殿時期，尤其是在希臘化壓迫與羅馬佔領的背景下，「彌賽亞」逐漸凝聚為一個末世論 (eschatological) 的專有概念——一位將要來的、被神特別揀選的拯救者。

---

## 二、第一世紀猶太人對彌賽亞的多元想像

非常重要的一點是：**第一世紀的猶太人對彌賽亞並沒有單一、統一的定義。** 不同群體有不同的期待：

### 1. 大衛後裔的政治-軍事君王（主流期待）

這是最廣泛的期待。根據撒母耳記下第 7 章的大衛之約，以及以賽亞書 11 章、耶利米書 23 章等先知預言，大部分猶太人期待一位：

- 出自大衛家族的合法君王
- 以軍事力量擊敗以色列的敵人（當時是羅馬帝國）
- 恢復以色列的政治獨立與領土完整
- 重建或淨化耶路撒冷聖殿
- 在耶路撒冷建立公義的統治，使以色列成為列國之首
- 帶來一個屬於「這個世界」的和平國度

《所羅門詩篇》(Psalms of Solomon) 第 17 篇（約主前一世紀）是這種期待的典型文獻：「主啊，求你為他們興起他們的王，大衛的子孫......使他以力量打碎不義的統治者......用鐵杖牧養耶和華的羊群。」

### 2. 祭司彌賽亞

死海古卷（昆蘭社群）的文獻顯示，愛色尼人期待**兩位**彌賽亞：一位是「亞倫的彌賽亞」（祭司），一位是「以色列的彌賽亞」（君王）。祭司彌賽亞的地位甚至高於君王彌賽亞。這反映了昆蘭社群對聖殿祭司體系純潔性的極度重視。

### 3. 超自然的天上人物

在《以諾一書》（特別是「比喻篇」，約主前一世紀至主後一世紀）中，出現了一位被稱為「人子」的超自然存在，祂在末日審判中扮演核心角色，坐在榮耀的寶座上審判列國。這個「人子」的形象源自但以理書 7:13-14，但在以諾傳統中被進一步發展為一位先存的 (pre-existent)、具有神聖屬性的人物。

### 4. 先知型的彌賽亞

有些人期待一位像摩西一樣的「末世先知」（參申 18:15-18），來做最終的啟示與教導。施洗約翰被問到「你是那先知嗎？」（約 1:21）就反映了這種期待。

### 5. 沒有彌賽亞的末世論

值得注意的是，有些猶太文獻的末世觀完全不包含彌賽亞人物——神自己直接介入歷史、審判列國、拯救以色列，不需要中介者。

## 三、第一世紀彌賽亞觀的共同特徵

儘管存在多元性，大多數彌賽亞期待有幾個共同要素：

1. **民族性**：彌賽亞是為以色列而來的，祂的使命首先是拯救以色列民族。
2. **政治性**：拯救包含具體的政治解放，而非僅僅「靈性」上的救贖。
3. **此世性**：彌賽亞國度是在這個地上實現的，不是「去天堂」。
4. **勝利性**：彌賽亞是勝利者，不是失敗者。祂擊敗敵人、建立國度。
5. **人性**：即使有超自然特質（如以諾傳統），彌賽亞基本上仍被理解為「人」，而非「神自己」。沒有任何主流第一世紀猶太文獻將彌賽亞等同於神（雅威）本身。

## 四、與今日教會「基督論」的主要差異

「基督」(Christos) 是 Mashiach 的希臘文翻譯，但經過新約神學與後續教義發展，這個概念已經與第一世紀猶太人的原始理解有非常大的距離：

| 面向 | 第一世紀猶太彌賽亞觀 | 今日教會基督論 |
|------|----------------------|----------------|
| **本質** | 人（被神揀選的人） | 神人二性——完全的神、完全的人（尼西亞信經，325 AD；迦克墩信經，451 AD） |
| **三位一體** | 完全沒有這個概念 | 基督是三位一體中的聖子，與父同質 (homoousios) |
| **先存性** | 少數文獻暗示（如以諾書），但非主流 | 核心教義——「太初有道」（約 1:1），基督是創世前就存在的 |
| **受苦與死亡** | 彌賽亞不應該受苦和死亡；這是失敗的標誌 | 十架受苦是核心——基督「必須」受苦（路 24:26），死亡是救贖計畫的核心 |
| **復活的意義** | 猶太人相信末日的普遍復活，但不是「彌賽亞先復活」 | 基督的復活是獨特的「初熟的果子」(firstfruits)，是救贖的證據和保證 |
| **使命範圍** | 以色列民族的政治解放與復興 | 全人類的靈性救贖——從罪與死亡中被拯救 |
| **救贖方式** | 透過軍事勝利與公義統治 | 透過代贖的死亡 (substitutionary atonement)——「背負世人的罪」 |
| **國度性質** | 地上的、政治的、此世的以色列王國 | 「已然-未然」(already/not yet) 的屬靈國度；最終的新天新地 |
| **再臨觀** | 不適用（彌賽亞還沒來過，何來「再臨」） | 基督會第二次降臨，帶來最終審判與新創造 |
| **原罪與贖罪** | 猶太教無「原罪」教義；人可以透過悔改和遵行律法與神和好 | 人因原罪與神隔絕，唯有透過基督的代贖才能得救 |

## 五、關鍵張力：「受苦的彌賽亞」問題

第一世紀猶太人最難以接受耶穌是彌賽亞的原因，就是**十字架**。

保羅自己說得很清楚：「我們卻是傳釘十字架的基督，在猶太人為絆腳石 (skandalon)」（林前 1:23）。

為什麼是絆腳石？因為：

- 被釘十字架是羅馬帝國對最卑賤罪犯的刑罰
- 申命記 21:23 說「被掛在木頭上的人是被神咒詛的」
- 彌賽亞應該**擊敗**羅馬，不是**被羅馬處死**
- 一個死在十字架上的人，在猶太彌賽亞的框架裡，就是一個**失敗的彌賽亞候選人**，句號

早期基督徒對此的神學回應是：重新詮釋舊約經文（特別是以賽亞書 53 章的「受苦的僕人」、詩篇 22 篇等），論證「受苦」本身就是彌賽亞使命的一部分。但這在當時是極具革命性的詮釋創新——在耶穌之前，幾乎沒有猶太文獻將以賽亞書 53 章的受苦僕人等同於彌賽亞。

---

## 六、小結

第一世紀猶太人的「彌賽亞」與今日教會的「基督」之間的差距，不只是程度上的不同，而是**範式性**的不同。猶太彌賽亞觀基本上是一個**政治-民族-此世**的概念；教會的基督論則是一個**宇宙性-靈性-超越性**的神學體系。

早期基督教的核心工作之一，就是在保留「彌賽亞」這個猶太詞彙的同時，徹底重新定義它的內涵——從「大衛的子孫要恢復以色列王國」，轉變為「神的獨生子透過十字架上的死亡拯救全人類脫離罪與死」。

這個轉變是否忠於原始的先知預言，還是根本性的重新詮釋？這正是猶太教與基督教兩千年來最核心的分歧之一。
