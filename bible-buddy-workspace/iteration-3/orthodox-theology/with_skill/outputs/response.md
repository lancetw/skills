# 東正教三大主張的第一世紀猶太評估：Theosis、聖像、聖傳

你一次提出了三個重大主張，每一個都有各自的經文根據。我會逐一從第一世紀猶太視角來檢驗，讓經文自己說話。

---

## 一、Theosis（神化）：彼得後書 1:4 說「與神的本性有分」？

### 經文

**彼得後書 1:3-4**（來源：Bible Gateway RCUV / FHL 呂振中 / 新漢語譯本）

> RCUV：「神的神能已把一切關乎生命和虔敬的事賜給我們……使我們既脫離世上從情慾來的敗壞，就得**分享神的本性**。」
>
> 呂振中：「使我們旣逃脫了世界上私慾裏的敗壞，就藉着這些恩得以**有分於上帝之神性**。」
>
> 新漢語譯本：「使你們藉着這些應許，得以脫離世上因私慾而來的敗壞，就可以**與神的本性有分**。」

希臘原文的關鍵詞是 θείας κοινωνοὶ φύσεως（theias koinōnoi physeōs），直譯為「神聖本性的分享者/參與者」。

### 第一世紀猶太視角

這裡必須先問一個關鍵問題：**第一世紀的猶太人會怎麼理解「分享神的本性」？**

**1. צֶלֶם（tselem）——神的形像**

**創世記 1:26-27**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：וַיֹּאמֶר אֱלֹהִים נַעֲשֶׂה אָדָם **בְּצַלְמֵנוּ כִּדְמוּתֵנוּ**
>
> RCUV：「神說：『我們要照着我們的**形像**，按着我們的**樣式**造人。』」

第一世紀的猶太理解：人被造時就帶有神的 צֶלֶם（tselem，形像）和 דְּמוּת（demut，樣式）。這是一種功能性的代表（functional representation）——人作為神在地上的代理人（vice-regent），管理受造界。這不是本體論上的等同（ontological identification），而是職分上的委派（vocational commission）。

**2. 詩篇 82:6 —— 「你們是諸神」**

**詩篇 82:6-7**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：אֲנִי אָמַרְתִּי **אֱלֹהִים אַתֶּם** וּבְנֵי עֶלְיוֹן כֻּלְּכֶם׃ אָכֵן **כְּאָדָם תְּמוּתוּן**
>
> RCUV：「我曾說：『你們是**諸神**，都是至高者的兒子。然而，你們要**死去，與世人一樣**。』」

耶穌本人在約翰福音 10:34-36 引用了這段經文：

> RCUV：「耶穌回答他們：『你們的律法書上不是寫着「我曾說你們是諸神」嗎？經上的話是不能廢的；如果那些領受神的道的人，神尚且稱他們為諸神……』」

但注意耶穌引用的方式——他在做一個 קַל וָחֹמֶר（kal va-chomer，從輕到重）的論證：如果連領受神話語的普通人都被稱為 אֱלֹהִים，那我自稱「神的兒子」有什麼褻瀆？這是一個辯論修辭，不是在教導「人可以在本質上成為神」。

而詩篇 82 本身的脈絡非常清楚——第 7 節立即宣告這些「諸神」**必死如凡人**。他們被稱為 אֱלֹהִים 是因為他們承擔了審判的職分（cf. 出埃及記 21:6，22:8-9 中 אֱלֹהִים 也被用來指「審判官」），而不是因為他們的本質發生了改變。

**3. 第一世紀猶太文獻中的「與神相近」概念**

在第二聖殿時期的文獻中，確實有「人被提升到天上」的概念——例如以諾（創世記 5:24）、以利亞（列王紀下 2:11）。死海古卷中的《安息日獻祭之歌》（Shirot Olat HaShabbat, 4Q400-407, ~150-100 BCE）描述天使崇拜的場景，暗示社群成員透過禮儀參與天上的敬拜。

但這些概念有一條清楚的紅線：**受造者永遠不會成為造物主。**

**以賽亞書 40:18, 25**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：וְאֶל מִי **תְּדַמְּיוּן** אֵל וּמַה דְּמוּת **תַּעַרְכוּ** לוֹ
>
> RCUV：「你們究竟將誰**比**神，用甚麼形像與他**相較**呢？」
>
> 40:25：「那聖者說：『你們將誰與我**相比**，與我**相等**呢？』」

以賽亞書的希伯來動詞 דמה（damah，比較/等同）和 ערך（arakh，排列/並比）在這裡建立了一道絕對的存有論邊界：YHWH 與一切受造物之間存在不可跨越的類別差異。

### 評估

彼得後書 1:4 的希臘文 κοινωνοὶ θείας φύσεως 所表達的，在第一世紀猶太語境中最可能的意思是：**透過遵行妥拉、脫離敗壞的行為，人恢復（或更充分地活出）神創造時所賦予的 צֶלֶם。** 這是一種倫理性的轉變（ethical transformation），不是本體性的融合（ontological union）。

東正教的 theosis 概念——特別是「人在本質上（κατ' οὐσίαν）與神聯合」的表述——要到第四世紀的亞他那修（Athanasius, ~296-373 CE）和卡帕多家教父（Cappadocian Fathers, 4th c.）才有系統性的發展。 ⚠ 這是一個後期教義發展，不能直接回讀到第一世紀文本中。

第一世紀的猶太人讀彼得後書 1:4，會理解為「活出神的品格、成為聖潔」，而不是「在本質上成為神」。

## 二、聖像（Icons）：基路伯不等於聖像？

### 經文

**出埃及記 20:4-5**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：לֹא **תַעֲשֶׂה** לְךָ **פֶסֶל** וְכׇל **תְּמוּנָה** ... לֹא **תִשְׁתַּחֲוֶה** לָהֶם וְלֹא **תׇעׇבְדֵם**
>
> RCUV：「不可為自己雕刻**偶像**（פֶּסֶל / pesel），也不可做甚麼**形像**（תְּמוּנָה / temunah）……不可跪拜那些像，也不可事奉它們。」

**出埃及記 25:18-20**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：וְעָשִׂיתָ שְׁנַיִם **כְּרֻבִים** זָהָב מִקְשָׁה תַּעֲשֶׂה אֹתָם מִשְּׁנֵי קְצוֹת הַכַּפֹּרֶת
>
> RCUV：「要造兩個用金子錘出的**基路伯**，從櫃蓋的兩端錘出它們。」

**申命記 4:12, 15-16**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文 (4:12)：וּ**תְמוּנָה** אֵינְכֶם רֹאִים **זוּלָתִי קוֹל**
>
> RCUV：「你們聽見說話的聲音，只有聲音，卻**沒有看見形像**。」
>
> 4:15-16：「你們為自己的緣故要分外謹慎；因為……你們**沒有看見任何形像**（תְּמוּנָה）。惟恐你們……為自己雕刻任何形狀的偶像（פֶּסֶל תְּמוּנַת כׇּל סָמֶל）。」

### 第一世紀猶太視角

東正教的論證邏輯是：「出埃及記 20 章禁止偶像，但 25 章命令製作基路伯，所以不是所有宗教圖像都被禁止。」這個論證在第一世紀猶太理解中經不起推敲，原因如下：

**1. 基路伯（כְּרוּבִים）的特殊性**

基路伯的製作有三個不可複製的條件：
- **直接命令**（מִצְוָה / mitzvah）：是 YHWH 親自指示的，不是人自行決定的（出 25:18）。
- **特定位置**：只在至聖所內，在約櫃蓋上（出 25:20）。
- **不被敬拜**：基路伯從來不是敬拜的對象。它們標示的是 YHWH 說話的位置——「我要在那裏與你相會，並要從法版之櫃的櫃蓋上，兩個基路伯的中間……」（出 25:22）。它們是 YHWH 臨在的建築標記，不是 YHWH 的形像。

**2. 申命記 4 章的禁令比出埃及記 20 章更嚴厲**

申命記 4:12 給出了禁止 תְּמוּנָה 的神學理由：**你們在何烈山只聽見聲音（קוֹל），沒有看見形像（תְּמוּנָה）。** 這直接排除了用任何視覺形像來代表 YHWH 的做法。重點不只是「不可拜偶像」，而是「YHWH 選擇不以視覺形像啟示自己」。

**3. 第一世紀的實際證據**

第一世紀猶太人對圖像的態度有大量歷史和考古證據：

- **約瑟夫**（Josephus, ~37-100 CE）記載，猶太人為了反對彼拉多將帶有凱撒像的軍旗帶入耶路撒冷，願意集體赴死（《猶太戰記》 II.9.2-3, ~75 CE）。
- **約瑟夫**也記載希律為耶路撒冷神殿建造的金鷹被猶太青年拆毀（《猶太古事記》 XVII.6.2-3, ~93 CE）。
- **死海古卷社群**（昆蘭）的生活空間中沒有發現任何宗教圖像。
- **早期猶太會堂**（如 Dura-Europos, ~244 CE——注意已是第三世紀）才開始出現聖經場景壁畫，而即使在那裡，YHWH 本身也不被描繪。

第一世紀的猶太共識是清楚的：**基路伯是妥拉中一個極特殊的命令，不能被推廣為「宗教圖像都可以」的一般原則。** 而東正教的聖像不只是裝飾，信徒向聖像鞠躬、親吻、焚香——在第一世紀猶太人看來，這恰恰是 הִשְׁתַּחֲוָה（hishtakhavah，俯伏/敬拜）和 עֲבוֹדָה（avodah，事奉/崇拜）——正是出埃及記 20:5 所禁止的動作。

### 評估

東正教從「基路伯」到「聖像可以」的推論，在第一世紀猶太邏輯中是無效的。基路伯是直接命令下的至聖所建築元素，不是信徒在家中或教堂中向之行禮的崇拜圖像。第一世紀的猶太人會對聖像實踐（鞠躬、親吻、焚香）感到極度不安，因為這些動作在妥拉的語境中就是 עֲבוֹדָה זָרָה（avodah zarah，異教崇拜）。

## 三、聖傳等同聖經（Holy Tradition = Scripture）？

### 經文

**帖撒羅尼迦後書 2:15**（來源：Bible Gateway RCUV / FHL 呂振中 / 新漢語譯本）

> RCUV：「弟兄們，你們要站立得穩，凡所領受的教導，無論是我們**口傳的**，是**信上寫的**，都要堅守。」
>
> 呂振中：「凡你們所受教的**傳統教訓**、無論是由我們**口傳**、或是由我們的**書信**所發表的，你們都要執守。」
>
> 新漢語譯本：「你們學到的教訓，無論是藉着我們的**言語**，還是藉着我們寫的**信**而學到的，都要持守。」

希臘原文中「傳統」是 παραδόσεις（paradoseis），對應希伯來文概念中的 מָסֹרֶת（masoret）。

東正教據此論證：保羅承認口傳傳統與書面經文有同等權威，所以後來的教會傳統也跟聖經一樣重要。

### 第一世紀猶太視角

**1. 耶穌對傳統（פָּרָדוֹסִיס / paradosis）的態度**

**馬可福音 7:5-9, 13**（來源：Bible Gateway RCUV）

> 法利賽人問：「你的門徒為甚麼不照**古人的傳統**？」
>
> 耶穌回答：「你們是離棄**神的誡命**，拘守**人的傳統**。」（7:8）
>
> 「這就是你們藉着繼承**傳統**，廢了**神的話**。」（7:13）

耶穌在這裡用的正是同一個希臘詞 παράδοσις（paradosis, 傳統）。他的立場極其明確：**當人的傳統與 YHWH 的妥拉衝突時，妥拉優先，傳統必須讓路。**

**2. 保羅在帖後 2:15 說的「傳統」是什麼？**

保羅這裡說的 παραδόσεις 是指他親自傳授的教導——他自己剛剛口頭講過的、或剛剛在信中寫下的東西。這是第一代使徒的直接教導，不是經過幾百年口耳相傳的累積傳統。

把「保羅對帖撒羅尼迦教會說的口頭教導」等同於「四世紀以後發展出的禮儀、教義、修道制度」，是一個巨大的邏輯跳躍。

**3. 妥拉對「增添」的禁止**

**申命記 4:2**（來源：Sefaria / Bible Gateway RCUV）

> 希伯來文：לֹא תֹסִפוּ **עַל הַדָּבָר** אֲשֶׁר אָנֹכִי מְצַוֶּה אֶתְכֶם וְלֹא תִגְרְעוּ מִמֶּנּוּ
>
> RCUV：「我吩咐你們的話，你們**不可加添**，也**不可刪減**。」

動詞 יָסַף（yasaf，加添）和 גָּרַע（gara，刪減）界定了啟示的完整性。第一世紀的猶太理解中，妥拉是 YHWH 啟示的核心和標準，一切解釋和傳統都必須接受妥拉的檢驗。

**4. 第一世紀猶太實際情況**

第一世紀的猶太世界確實有強大的口傳傳統——法利賽人教導 תּוֹרָה שֶׁבְּעַל פֶּה（Torah she-b'al peh，口傳妥拉）。但即使在法利賽傳統內部，也存在關於口傳傳統權威性的辯論。撒都該人（Sadducees）只接受書面妥拉（Written Torah）。昆蘭社群有自己的解經傳統（pesher），但不認為它等同經文本身。

而耶穌的立場——從馬可福音 7 章來看——是明確的：**傳統在服務妥拉時有用，傳統凌駕妥拉時有害。** 傳統必須始終受妥拉的審核。

### 評估

保羅在帖後 2:15 確實肯定了口傳教導的重要性，但他指的是自己剛傳遞的第一代使徒教導，不是幾個世紀後累積的教會傳統體系。而耶穌在馬可福音 7 章建立了一個清楚的原則：**神的話（דְּבַר יְהוָה）高於人的傳統（παράδοσις τῶν ἀνθρώπων）。** 「聖傳等同聖經」這個主張在第一世紀猶太框架中沒有根據。

## 總結：三個主張的第一世紀猶太評估

| 主張 | 東正教論據 | 第一世紀猶太評估 |
|------|-----------|----------------|
| **Theosis（神化）** | 彼得後書 1:4「與神的本性有分」 | 第一世紀理解為倫理轉變（恢復 צֶלֶם），不是本體融合。「成為神」的系統教義是 4 世紀發展。以賽亞書 40:18, 25 設立了不可跨越的存有論界線。 |
| **聖像崇敬** | 出埃及記 25 章的基路伯 | 基路伯是直接命令下的至聖所建築元素，不被敬拜。申命記 4:12-16 明確禁止為 YHWH 製作任何 תְּמוּנָה。約瑟夫記載第一世紀猶太人為反對宗教圖像願意集體赴死。 |
| **聖傳 = 聖經** | 帖後 2:15「口傳的…信上寫的」 | 保羅指第一代使徒的直接教導，不是後世累積的傳統。耶穌在馬可 7:8-13 明確將「神的誡命」置於「人的傳統」之上。申命記 4:2 禁止增添。 |

三個主張都是後期教會發展出的教義，各自有其神學內部邏輯，但都無法從第一世紀猶太文本和歷史證據中得到支持。
