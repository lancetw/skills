# 「耶穌學密宗」的說法站得住腳嗎？——從第一世紀猶太背景檢視三個類比

## 前言：「失蹤的十八年」從何而來？

在進入經文分析之前，必須先處理這個理論的根基——「耶穌去西藏學密宗」這個說法。

這個說法最早來自俄國記者 **Nicolas Notovitch** 於 1894 年出版的 *The Unknown Life of Jesus Christ*。他聲稱在拉達克（Ladakh）的赫密斯寺（Hemis Monastery）發現一份名為《聖伊薩傳》（*Life of Saint Issa*）的藏文手稿，記載耶穌青年時期到印度和西藏學佛。然而：

- 語言學家 **Max Müller** 當時就指出 Notovitch 可能被惡作劇愚弄，或根本自行捏造。
- 一位英國女性訪問赫密斯寺後，住持明確表示：**「整個故事沒有一個字是真的。沒有任何俄國人來過這裏。過去五十年內沒有任何人因斷腿而被收留。」**
- 主流學者共識：**Marcus Borg** 指出耶穌曾旅行至埃及或印度的說法「毫無歷史根基」（without historical foundation）。**John Dominic Crossan**（Jesus Seminar 聯合創辦人）表示，所有試圖填補這十五至十八年空白的理論，「沒有任何一個得到現代學術的支持」。

福音書對耶穌十二歲到三十歲之間的沉默，在第一世紀猶太文化中完全正常——古代傳記不關心「成長歷程」。路加福音 2:51-52 簡單交代他回到拿撒勒，「服從父母」，這暗示他像任何加利利猶太青年一樣，跟隨父親約瑟做木匠（τέκτων / tekton），在會堂學習妥拉。

**結論：「耶穌去西藏」的前提本身就沒有任何歷史證據。但即使暫時擱置這個前提，以下三個具體的「密宗類比」在第一世紀猶太背景下也完全站不住腳。**

## 一、登山變像（馬太福音 17:1-8）≠ 虹光身（Rainbow Body / འཇའ་ལུས / jalü）

### 經文

**和合本修訂版 (RCUV 2010)：**
> [1] 過了六天，耶穌帶著彼得、雅各和雅各的弟弟約翰，領他們悄悄地上了高山。[2] 他在他們面前變了形像，他的臉明亮如太陽，衣裳潔白如光。[3] 忽然，有摩西和以利亞向他們顯現，與耶穌說話。[4] 彼得回應，對耶穌說：「主啊，我們在這裏真好！你若願意，我就在這裏搭三座棚，一座為你，一座為摩西，一座為以利亞。」[5] 說話之間，忽然有一朵明亮的雲彩遮蓋他們，又有聲音從雲彩裏出來，說：「這是我的愛子，我所喜愛的。你們要聽從他！」[6] 門徒聽見，就俯伏在地，極其害怕。[7] 耶穌進前來，拍拍他們，說：「起來，不要害怕！」[8] 他們舉目，不見一人，只見耶穌獨自一人。
>
> （來源：Bible Gateway RCUV / 信望愛 RCUV）

**呂振中譯本（最忠於原文的中文譯本）：**
> [2] 在他們面前他變了形質：他的臉面發光像日頭，他的衣裳變白如光一樣。
>
> （來源：信望愛 FHL，呂振中譯本）

### 這段經文的第一世紀猶太背景

登山變像的每一個元素都直接回溯到**妥拉**（Torah）和**先知書**（Nevi'im），完全不需要任何外部框架來解釋：

#### (a) 摩西在西奈山的臉發光——קָרַן / qaran

**出埃及記 34:29（希伯來原文，來源：Sefaria）：**
> וּמֹשֶׁה לֹא־יָדַע כִּי קָרַן עוֹר פָּנָיו בְּדַבְּרוֹ אִתּוֹ
>
> "摩西不知道自己的臉上的皮膚因（耶和華）和他說話而 *qaran*。"

**和合本修訂版：**
> 摩西不知道自己臉上的皮膚因耶和華和他說話而**發光**。
>
> （來源：Bible Gateway RCUV）

希伯來動詞 **קָרַן**（qaran）的字根與 **קֶרֶן**（qeren，「角」）相同。七十士譯本（LXX）譯為 δεδόξασται（dedoxastai，「被榮耀化」），將此理解為神聖榮光（כָּבוֹד / kavod）的反射。耶柔米（Jerome）誤譯為「有角」（cornuta），導致米開朗基羅的摩西像有角——但這是翻譯錯誤，不是密宗虹光。

**關鍵差異：** 摩西的臉發光是因為他進入了 YHWH 的臨在、與 YHWH 對話之後的**被動結果**——他自己「不知道」。這是「與神面對面」（פָּנִים אֶל־פָּנִים / panim el panim）的後果，不是任何修行技術的成就。

#### (b) 西奈山神顯現（theophany）的整套元素

| 登山變像元素 | 西奈山出處 | 經文 |
|-------------|-----------|------|
| 高山 | 西奈山 / 何烈山 | 出 24:15-18 |
| 六天後 | 雲彩遮蓋山六天，第七天呼召摩西 | 出 24:16 |
| 臉面發光 | 摩西的臉 qaran | 出 34:29 |
| 明亮的雲彩 | 耶和華的榮耀（כְּבוֹד יְהֹוָה / kevod YHWH）如烈火 | 出 24:16-17 |
| 從雲彩中的聲音 | YHWH 從雲中說話 | 出 24:16 |
| 摩西出現 | 摩西本人——妥拉的頒佈者 | — |
| 以利亞出現 | 以利亞在何烈山（同一座山）經歷神顯現 | 王上 19:8-13 |
| 門徒俯伏害怕 | 以西結見神的榮耀（kavod）後俯伏在地 | 結 1:28 |
| 搭三座棚（sukkot） | 住棚節（Sukkot）的帳棚 | 利 23:42-43 |

馬太福音的作者刻意讓讀者看到：**耶穌就是新的摩西，這座山就是新的西奈山。** 學者 Andrei Orlov 在〈Which Moses? Jewish Background of Jesus' Transfiguration〉中指出，在第二聖殿時期的猶太文獻裏，摩西本人已經被描繪為吸收了神聖榮光（kavod）的特質。馬太福音的描述正是在這個猶太傳統中運作。

#### (c) 但以理書 12:3——義人發光如星

**但以理書 12:3（希伯來原文，來源：Sefaria）：**
> וְהַמַּשְׂכִּלִים יַזְהִרוּ כְּזֹהַר הָרָקִיעַ וּמַצְדִּיקֵי הָרַבִּים כַּכּוֹכָבִים לְעוֹלָם וָעֶד
>
> "通達人（maskilim）必**發光**（yazhiru）如穹蒼的光輝（zohar）；使多人歸義的，必發光如星，直到永永遠遠。"

動詞 **יַזְהִרוּ**（yazhiru）來自字根 ז-ה-ר（z-h-r，發光、放光），名詞 **זֹהַר**（zohar，光輝）。這是末世復活的異象——義人在來世會像星辰發光。這是一世紀猶太人理解「發光」的框架。

### 虹光身（jalü / འཇའ་ལུས）是什麼？為何不能類比？

虹光身（jalü）是藏傳佛教**大圓滿**（Dzogchen）傳統中的概念，與**立斷**（trekchö）和**頓超**（tögal）修行有關，指修行者在圓寂時肉體縮小或化為光。這個概念：

- 源於西藏本土的**苯教**（Bon），是前佛教的（pre-Buddhist）傳統。
- 與八世紀的蓮花生大師（Padmasambhava）相關聯。
- 最早的文獻出現在**七至九世紀**的密續文本中。
- 描述的是修行者經過長期特定修行後在**死亡時**身體的轉化。

| 比較項目 | 登山變像 | 虹光身 |
|---------|---------|--------|
| 時間點 | 生前，短暫事件 | 死亡時或圓寂後 |
| 肉體狀態 | 變像後恢復正常，門徒看到「只見耶穌獨自一人」 | 肉體永久消失或縮小 |
| 原因機制 | 神的榮耀（kavod）的顯現 | 長期大圓滿修行的果位 |
| 文化背景 | 西奈山神顯現傳統 | 苯教 + 藏傳佛教大圓滿 |
| 時代 | 第一世紀巴勒斯坦猶太教 | 七至九世紀西藏 |
| 見證者反應 | 「俯伏在地，極其害怕」——典型的聖經神顯反應 | 弟子見證，但反應框架完全不同 |

**年代問題無法迴避：** 虹光身的概念最早出現在耶穌之後六百至八百年。即使兩者表面上都涉及「身體發光」，也不能倒因為果地說前者來自後者。

## 二、「你們在我裏面，我也在你們裏面」（約翰福音 14:20）≠ 上師瑜伽（Guru Yoga）

### 經文

**和合本修訂版 (RCUV 2010)：**
> [15] 「你們若愛我，就會遵守我的命令。[16] 我要求父，父就賜給你們另外一位保惠師，使他永遠與你們同在。[17] 他就是真理的靈，是世人不能接受的。因為他們既看不見他，也不認識他；你們卻認識他，因他常與你們同在，也要在你們裏面。[18] 我不會撇下你們為孤兒，我必到你們這裏來。[19] 再過不久，世人不再看見我，你們卻會看見我，因為我活着，你們也要活着。**[20] 到那日，你們就會知道我在父裏面，你們在我裏面，我也在你們裏面。**[21] 有了我的命令而又遵守的人，就是愛我的；愛我的人，我父要愛他，我也要愛他，並且要親自向他顯現。」[22] 猶大（不是加略人猶大）問耶穌：「主啊，為甚麼親自向我們顯現，而不向世人顯現呢？」[23] 耶穌回答他說：「凡愛我的人就會遵守我的道，我父也會愛他，並且我們要到他那裏去，**與他同住**。」[24] 不愛我的人就不遵守我的道。你們所聽見的道不是我的，而是差我來之父的。
>
> （來源：Bible Gateway RCUV / 信望愛 RCUV）

**呂振中譯本：**
> [20] 到那日、你們就知道我在我父裏面，你們在我裏面，我也在你們裏面。
> [23] ……我們也必來找他，**將他做我們的住處**。
>
> （來源：信望愛 FHL，呂振中譯本）

注意呂振中的翻譯更忠實地呈現了希臘文 μονὴν παρ᾽ αὐτῷ ποιησόμεθα（monēn par' autō poiēsometha）的意涵——「我們會在他那裏建立居所」。

### 第一世紀猶太背景：שְׁכִינָה / Shekhinah（同在）

要理解「你們在我裏面，我也在你們裏面」，第一世紀猶太人會立刻想到的不是任何「上師與弟子合一」的神秘修行，而是一個他們每天活在其中的核心概念：**שְׁכִינָה**（Shekhinah）——神的**同在**、**居住**。

- 字根 **שׁ-כ-נ**（sh-k-n）意為「居住、安頓」，同一字根出現在 **מִשְׁכָּן**（mishkan，「會幕」——神在以色列中居住的帳幕）。
- 約翰福音 1:14 的「道成了肉身，**住在**我們中間」（ἐσκήνωσεν / eskēnōsen），希臘文本身就在回響希伯來文的「搭帳幕」（sh-k-n）。
- 拉比傳統說：「當兩個人坐在一起研讀妥拉的話語時，Shekhinah 就在他們中間。」（Mishnah Avot 3:2，雖編纂於約 200 CE，但反映的是更早的口傳傳統。）

約翰福音 14:20 的「在……裏面」（ἐν / en）描述的是**盟約關係**（covenant relationship）中的相互歸屬，不是「本體合一」（ontological merger）。觀察上下文：

1. **14:15** —— 「你們若**愛**我，就會**遵守我的命令**」：這是盟約語言（covenant language）。在妥拉中，「愛」（אָהַב / ahav）YHWH 就是「遵守祂的誡命」（申 6:5-6）。
2. **14:23** —— 「凡愛我的人就會遵守我的道……我們要到他那裏去，**與他同住**」：「同住」（μονή / monē）回溯到會幕 / 聖殿的概念——神在以色列中**住下來**。
3. **14:24** —— 「你們所聽見的道不是我的，而是**差我來之父的**」：耶穌明確區分自己與父——他是被差遣者，不是與門徒融合為一。

### 上師瑜伽（Guru Yoga）是什麼？為何不能類比？

上師瑜伽是藏傳佛教金剛乘的核心修行，修行者**觀想上師為佛**，將自己的身、語、意**融入**上師的證悟功德。

- 作為獨立修行體系出現在**十一至十二世紀的西藏**。
- 建立在印度密續中的上師崇敬（guru-bhakti）傳統上，但印度密續本身最早可追溯到**七世紀 CE**。
- 核心機制：修行者主動觀想自身與上師**合而為一**，消除主客對立。

| 比較項目 | 約翰福音 14:20 | 上師瑜伽 |
|---------|---------------|---------|
| 關係結構 | 盟約關係：「愛我就遵守命令」 | 觀想合一：消除主客二元 |
| 「在……裏面」的意涵 | Shekhinah 同在——神住在祂百姓中間 | 本體論的融合——弟子即上師即佛 |
| 中介 | 「真理的靈」（14:17）——被差來的 | 無中介，直接合一 |
| 區分是否保留 | 是——「道不是我的，是父的」（14:24） | 否——合一後無分別 |
| 實踐方式 | 遵守誡命、愛（covenant obedience） | 觀想修行（visualization practice） |
| 文化世界 | 第一世紀巴勒斯坦猶太盟約神學 | 十一至十二世紀西藏密續傳統 |

**最關鍵的差異：** 約翰福音 14:20 的上下文是**盟約忠誠**（covenant faithfulness）——「愛我 → 遵守命令 → 我們會來同住」。這是妥拉的邏輯（申命記 6:4-9），不是修行技術的邏輯。在上師瑜伽中，「合一」是修行的**目標和方法**；在約翰福音中，「同在」是忠於盟約的**結果**。

## 三、治病趕鬼 ≠ 密宗灌頂加持（Abhisheka / དབང / wang）

### 經文

**馬太福音 12:27-28（RCUV）：**
> [27] 我若靠着別西卜趕鬼，你們的子弟趕鬼又靠着誰呢？這樣，他們要作你們的判官。[28] 我若靠着**神的靈**趕鬼，那麼，**神的國就已臨到你們了**。
>
> （來源：Bible Gateway RCUV）

**以賽亞書 61:1（希伯來原文，來源：Sefaria）：**
> רוּחַ אֲדֹנָי יֱהֹוִה עָלָי יַעַן מָשַׁח יְהֹוָה אֹתִי לְבַשֵּׂר עֲנָוִים
>
> "主 YHWH 的靈在我身上，因為 YHWH **膏了**（mashach / מָשַׁח）我，去向困苦人傳好消息。"

**路加福音 4:18-21（RCUV）：**
> [18]「主的靈在我身上，因為他用**膏**膏我，叫我傳福音給貧窮的人；差遣我宣告：被擄的得釋放，失明的得看見，受壓迫的得自由，[19] 宣告神悅納人的禧年。」[20] 於是他把書捲起來，交還給管理人，就坐下。[21] 耶穌對他們說：「你們聽見的這段經文，**今天已經應驗了**。」
>
> （來源：Bible Gateway RCUV）

### 第一世紀猶太背景：趕鬼和治病完全是猶太本土傳統

#### (a) 猶太驅邪傳統

第一世紀的猶太世界裏，趕鬼是一個**已知的猶太實踐**，完全不需要引入任何外部來源：

- **約瑟夫**（Josephus，約 93 CE 寫作）在《猶太古史》（Antiquities 8.46-48）記載他親眼見到一位名叫 **Eleazar** 的猶太人在維斯帕先（Vespasian）面前驅鬼。Eleazar 使用所羅門的咒語（incantation）、帶有特定草藥根的戒指放在被附者鼻前，將鬼從鼻孔引出。
- **死海古卷**（Dead Sea Scrolls，約 2 世紀 BCE — 1 世紀 CE）包含多篇驅邪禱文和咒語，如 *4Q510-511*（Songs of the Sage）、*11Q11*（Apocryphal Psalms），專門對付致病的惡靈。
- 《多俾亞傳》（Tobit，約 2 世紀 BCE）記載天使拉斐爾教導多俾亞用魚的心和肝來驅逐惡魔 Asmodeus。
- **耶穌自己指出猶太同儕也在趕鬼**——馬太福音 12:27：「你們的子弟趕鬼又靠著誰呢？」這證明趕鬼不是他獨有的，而是第一世紀猶太文化中已有的實踐。

#### (b) 先知治病傳統

耶穌的治病直接延續妥拉和先知書中的傳統：

- **以利沙**（Elisha）治好亞蘭元帥乃縵的皮膚病（列王紀下 5:1-14）——方式是在約旦河浸泡七次。不是灌頂，是先知以 YHWH 之名發出指令。
- **以利亞**（Elijah）使寡婦之子復活（列王紀上 17:17-24）——是向 YHWH 呼求禱告。
- **摩西**舉銅蛇治癒被蛇咬的以色列人（民數記 21:8-9）——是遵從 YHWH 的直接指示。

#### (c)「膏」（mashach）≠「灌頂」（abhisheka / wang）

耶穌在路加福音 4:18 引用以賽亞書 61:1 時，用的是 **מָשַׁח**（mashach，「膏」）——同一字根正是 **מָשִׁיחַ**（mashiach，「受膏者」，即「彌賽亞」）的來源。

在以色列傳統中，「膏」是：
- 用油倒在頭上，表示**被 YHWH 揀選委任**執行特定任務（王、祭司、先知）。
- **撒母耳膏大衛**（撒上 16:13）：「從那日起，耶和華的靈大大感動大衛。」

密宗灌頂（abhisheka / wang）則是：
- 密續上師在壇城（mandala）中對弟子進行的**祕密儀式**，傳授特定本尊的修法。
- 最早的密續文本（如《密集金剛密續》/ Guhyasamāja Tantra）出現在**四至七世紀 CE**。「灌頂」作為體系化的密續修行，最早在**七世紀 CE** 才有文獻記載。

| 比較項目 | 猶太受膏（mashach） | 密宗灌頂（abhisheka） |
|---------|-------------------|---------------------|
| 執行者 | 先知代表 YHWH 膏立 | 金剛上師（vajra guru）傳授 |
| 目的 | 委任執行盟約中的角色（王、祭司、先知） | 授權修行特定本尊法門 |
| 框架 | 盟約神學——YHWH 與以色列 | 密續傳承——上師與弟子的秘密傳授 |
| 媒介 | 橄欖油（שֶׁמֶן / shemen） | 水、花、寶瓶等密續法器 |
| 公開性 | 通常公開（撒母耳膏掃羅、膏大衛） | 必須秘密進行，有嚴格三昧耶戒 |
| 時代 | 自妥拉時代起（約公元前十三世紀以降） | 約七世紀 CE 起 |

## 方法論問題：表面相似 ≠ 歷史關聯

將登山變像比作虹光身、將「在我裏面」比作上師瑜伽、將治病比作灌頂加持，犯的是同一個邏輯錯誤：**以表面形態的相似性推論歷史因果關係**。這在宗教學中稱為 *parallelomania*（「平行狂熱」），由學者 Samuel Sandmel 於 1962 年提出批評。

具體問題：

1. **年代倒置**：密宗的所有相關概念（虹光身、上師瑜伽、灌頂加持）最早出現在七至十二世紀 CE，比耶穌晚了六百到一千年。要說耶穌「學了」這些，等於說他學了一個在他死後六百年才出現的東西。

2. **忽略本土背景**：每一個被拿來類比的元素，在第一世紀猶太傳統中都有**完整且充分的解釋**——西奈山神顯現、Shekhinah 同在、先知治病驅邪。不需要引入任何外部來源。歷史方法的基本原則是：**當本土傳統已能充分解釋時，不需假設外來影響。**

3. **去脈絡化**：這些類比從兩個完全不同的宗教體系中各抽取一個元素進行比較，完全忽略了各自的**整體脈絡**。登山變像的意義在西奈山盟約敘事中；約翰福音 14:20 的意義在盟約忠誠的框架中；治病趕鬼的意義在先知傳統和神國臨到的末世框架中。抽離脈絡後，任何宗教都可以跟任何宗教產生「相似」。

---

## 總結

從第一世紀猶太背景來看，這三個類比**完全站不住腳**：

| 聲稱 | 實際的第一世紀猶太背景 | 密宗概念最早年代 |
|------|----------------------|----------------|
| 登山變像 = 虹光身 | 西奈山神顯現（出 24, 34）；kavod 傳統 | 七至九世紀 CE |
| 約翰 14:20 = 上師瑜伽 | Shekhinah 同在；盟約語言 | 十一至十二世紀 CE |
| 治病趕鬼 = 灌頂加持 | 先知傳統（以利亞/以利沙）；猶太驅邪傳統 | 七世紀 CE |

這些經文不需要密宗來解釋，因為它們在自己的猶太世界裏已經完全有意義。把密宗投射到第一世紀猶太文本上，不是在理解耶穌——是在用一千年後的框架覆蓋他自己的世界。
