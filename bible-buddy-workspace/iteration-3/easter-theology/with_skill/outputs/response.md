# 復活節（Easter）vs 逾越節（Pesach）：從第一世紀猶太脈絡看耶穌的復活

你問了三個核心問題。我會逐一處理，每個都回到經文原文和歷史證據。

---

## 一、耶穌的復活跟逾越節有什麼關係？

### 答案：關係不是間接的——復活事件本身就發生在逾越節期間，而且最早的門徒是用逾越節的神學框架來理解復活的。

### 1. 逾越節的時間結構

Torah 規定了一組彼此相連的節期（מוֹעֲדִים / mo'adim，「指定的時間」）：

| 節期 | 日期 | Torah 出處 |
|------|------|-----------|
| **逾越節 / Pesach** (פֶּסַח) | 尼散月14日黃昏 | 出埃及記 12:6；利未記 23:5 |
| **除酵節 / Chag HaMatzot** (חַג הַמַּצּוֹת) | 尼散月15-21日 | 利未記 23:6-8 |
| **初熟節 / Yom HaBikkurim** (יוֹם הַבִּכּוּרִים) | 除酵節期間安息日的次日 | 利未記 23:9-14 |

**經文**（來源：Sefaria / Bible Gateway RCUV）：

> **利未記 23:5-6, 10-11**
>
> 希伯來文：בַּחֹ֣דֶשׁ הָרִאשׁ֗וֹן בְּאַרְבָּעָ֥ה עָשָׂ֛ר לַחֹ֖דֶשׁ בֵּ֣ין הָעַרְבָּ֑יִם פֶּ֖סַח לַיהֹוָֽה׃ וּבַחֲמִשָּׁ֨ה עָשָׂ֥ר יוֹם֙ לַחֹ֣דֶשׁ הַזֶּ֔ה חַ֥ג הַמַּצּ֖וֹת לַיהֹוָ֑ה
>
> RCUV：「正月十四日黃昏的時候，是向耶和華守的逾越節。這月的十五日是向耶和華守的除酵節；你們要吃無酵餅七日。」
>
> 23:10-11：「你要吩咐以色列人，對他們說：你們到了我賜給你們的地，收割莊稼的時候，要把初熟莊稼中的一捆拿來給祭司。他要把這捆在耶和華面前搖一搖，使你們蒙悅納。祭司要在安息日的次日把這捆搖一搖。」

關鍵希伯來詞：

- **עֹמֶר** (omer) —「一捆」，初熟大麥的搖祭
- **מִמׇּחֳרַת הַשַּׁבָּת** (mimahorat ha-Shabbat) —「安息日的次日」——這個詞組在第一世紀引發了法利賽人與撒都該人的爭論：法利賽人認為「安息日」指的是除酵節第一天（節期安息日），撒都該人和昆蘭社群認為指的是七日安息日（שַׁבָּת，Shabbat）。若按撒都該人的理解，初熟節永遠落在「七日的第一天」——也就是後來所說的「星期日」。

### 2. 福音書的敘事：耶穌的死亡和復活完全嵌在逾越節的時間框架中

**路加福音 22:7-8, 15**（RCUV）：
> 「除酵節到了，這一天必須宰逾越節的羔羊。耶穌打發彼得和約翰，說：『你們去為我們預備逾越節的宴席，好讓我們吃。』」
>
> 「我非常渴望在受害以前和你們吃這逾越節的宴席。」

**馬太福音 26:17-18**（RCUV）：
> 「除酵節的第一天，門徒來問耶穌：『你要我們在哪裏給你預備吃逾越節的宴席呢？』耶穌說：『你們進城去，到某人那裏，對他說：老師說：我的時候快到了，我要和我的門徒在你家裏守逾越節。』」

**約翰福音 19:14**（RCUV）：
> 「那日是逾越節的預備日，約在正午。彼拉多對猶太人說：『看哪，你們的王！』」

注意約翰福音的時間線：約翰把耶穌的死亡時刻與聖殿中宰殺逾越節羔羊的時刻對齊。這不是巧合——約翰在進行有意的神學編排。

**約翰福音 19:31-33, 36**（RCUV）：
> 「因為這日是預備日，又因為那安息日是個大日子，猶太人就來求彼拉多叫人打斷他們的腿，把他們搬走......當他們來到耶穌那裏，見他已經死了，就沒有打斷他的腿。......這些事發生，為要應驗經上的話：『他的骨頭一根也不可折斷。』」

約翰引用的「經上的話」出自兩處 Torah：

> **出埃及記 12:46**（Sefaria）：וְעֶ֖צֶם לֹ֥א תִשְׁבְּרוּ־בֽוֹ
>
> RCUV：「骨頭一根也不可折斷。」

> **民數記 9:12**（Sefaria）：וְעֶ֖צֶם לֹ֣א יִשְׁבְּרוּ־ב֑וֹ כְּכׇל־חֻקַּ֥ת הַפֶּ֖סַח יַעֲשׂ֥וּ אֹתֽוֹ
>
> RCUV：「羔羊的骨頭一根也不可折斷。他們要照逾越節的一切律例守這節。」

也可能同時呼應了：

> **詩篇 34:21**（希伯來文編號；中文 34:20）（Sefaria）：שֹׁמֵ֥ר כׇּל־עַצְמוֹתָ֑יו אַחַ֥ת מֵ֝הֵ֗נָּה לֹ֣א נִשְׁבָּֽרָה
>
> RCUV：「又保護他全身的骨頭，連一根也不折斷。」

### 3. 保羅的理解：復活 = 初熟的果子（Firstfruits）

保羅——一個受過訓練的法利賽人——用兩個逾越節期間的核心概念來解釋耶穌的死和復活：

**哥林多前書 5:7-8**

| 譯本 | 經文 |
|------|------|
| RCUV | 「因為我們逾越節的羔羊——基督已經被殺獻為祭牲了。所以，我們來守這節，不可用舊酵，就是不可用惡毒、邪惡的酵，只用純潔真實的無酵餅。」 |
| 新漢語譯本 | 「因為我們的逾越節羔羊基督，已經被殺獻為祭了。所以我們守這節，不可用舊酵，也不可用惡毒邪惡的酵，要用純潔真誠的無酵餅。」 |
| 呂振中 | 「因爲我們逾越節的羔羊也已經宰獻爲祭了，就是基督。所以我們守這節、不可用舊酵，也不可用惡毒邪惡之酵，卻要用純潔眞誠之無酵餅。」 |

注意保羅說「我們來守這節」（ἑορτάζωμεν / heortazōmen）——他寫信給哥林多的外邦信徒時，仍然使用逾越節 / 除酵節的語言，視之為理所當然的框架。希臘文 τὸ πάσχα ἡμῶν ἐτύθη Χριστός（我們的逾越節已被宰殺了，就是基督）——πάσχα 直接音譯自亞蘭文 פַּסְחָא (pasha)，源自希伯來文 פֶּסַח (pesach)。

**哥林多前書 15:20, 23**

| 譯本 | 經文 |
|------|------|
| RCUV | 「其實，基督已經從死人中復活，成為睡了之人初熟的果子。......初熟的果子是基督；然後在他來的時候，是那些屬於基督的。」 |
| 新漢語譯本 | 「但如今，基督已經從死人中復活，成為睡了的人初熟的果子。......初熟的果子是基督，然後在他來臨的時候，是那些屬於基督的人。」 |
| 呂振中 | 「其實不然，基督已經從死人中得甦活起來、做長眠着之人的初熟果子了。......初熟果子是基督；後來是屬基督的，就在他御臨的時候。」 |

保羅用的希臘文 ἀπαρχή (aparchē,「初熟的果子」) 對應的就是利未記 23:10 的 רֵאשִׁית קְצִירְכֶם (reshit qetzirekhem,「你們收割的初熟」)——逾越節期間的初熟節搖祭。保羅的意思是：**耶穌的復活就是初熟節的實現**——第一捆被搖在 YHWH 面前的大麥，預示著整片田地的收成（全體死人的復活）。

### 第一世紀的結論

對第一世紀的猶太門徒來說，耶穌的復活不是一個獨立的事件，而是 Torah 節期日曆（利未記 23 章）中三個相連節期的完成：

1. **逾越節**（尼散月14日）→ 耶穌作為逾越節羔羊被殺
2. **除酵節**（尼散月15-21日）→ 除去舊酵，進入新的無酵生活
3. **初熟節**（除酵節期間安息日的次日）→ 耶穌從死裡復活，成為 ἀπαρχή / רֵאשִׁית

這三者在第一世紀是**同一組節期**，不是分開的事件。

## 二、為什麼教會不在逾越節紀念復活？

### 答案：這是一個歷史過程，涉及政治權力、族群身分和神學重新定義。以下是時間線。

### 1. 第一到第二世紀：早期信徒在逾越節紀念

使徒行傳的作者仍然使用逾越節作為時間標記：

> **使徒行傳 12:3-4**（RCUV）：「他見猶太人喜歡這事，也去拿住彼得。那時候正是除酵節期間。希律捉了彼得，押在監裏......企圖要在逾越節後把他提出來，當着百姓辦他。」

最早的信徒——全部是猶太人——自然在逾越節紀念耶穌的死與復活。這不是「選擇」，而是他們的正常生活節奏。

### 2. 第二世紀：Quartodeciman 爭議（十四日派爭議）

「Quartodeciman」來自拉丁文 *quartadecima*（第十四），指的是在尼散月14日（逾越節當天）紀念耶穌受難的做法。

- **約 155 CE**：小亞細亞的示每拿主教 **坡旅甲**（Polycarp）訪問羅馬，與羅馬主教 **安尼西圖**（Anicetus）討論此事。坡旅甲堅持他的做法來自使徒約翰的傳承。兩人無法達成一致，但維持了團契。
- **約 195 CE**：羅馬主教 **維克多一世**（Victor I）升高衝突，威脅要將小亞細亞的十四日派教會逐出教會。以弗所主教 **坡利克拉底**（Polycrates）回信堅持使徒傳統。里昂主教 **愛任紐**（Irenaeus）寫信勸維克多不要破裂。
- 這些歷史記載保存在優西比烏（Eusebius）的《教會史》（*Historia Ecclesiastica*，約 325 CE）中。

核心爭議：小亞細亞的教會跟隨猶太曆法，在尼散月14日紀念；羅馬及其他教會改為在逾越節後的星期日紀念。

### 3. 325 CE：尼西亞會議的決定

尼西亞會議確立了兩條規則：

1. **脫離猶太曆法的依賴**：不再根據猶太拉比計算的尼散月14日來決定日期
2. **統一為星期日**：復活紀念定在春分後第一個滿月之後的第一個星期日

會議的書信（保存在優西比烏的記載中）明確表示：不要與「猶太人一起」（μετά，meta）慶祝。這個決定的動機不純粹是神學性的——它反映了第四世紀教會與猶太社群的政治分離。

### 4. 結果

尼西亞的公式（春分 → 滿月 → 星期日）與猶太曆的尼散月14日之間**有時會重合，但更常不一致**。這不是偶然——會議的目的之一就是要切斷與猶太節期日曆的連結。

需要注意的是：今天大多數語言中「復活節」的名稱仍然保留了逾越節的痕跡——

| 語言 | 名稱 | 來源 |
|------|------|------|
| 希臘文 | Πάσχα (Pascha) | 亞蘭文 פַּסְחָא |
| 拉丁文 | Pascha | 同上 |
| 法文 | Pâques | 同上 |
| 西班牙文 | Pascua | 同上 |
| 義大利文 | Pasqua | 同上 |
| 希伯來文 | פֶּסַח (Pesach) | Torah 原文 |

只有英文 **Easter** 和德文 **Ostern** 使用了不同的詞源。

## 三、Easter 這個名字來自 Ishtar 嗎？兔子和彩蛋在第一世紀有根據嗎？

### 1. Easter 的語源：不是 Ishtar

這是一個在網路上廣泛流傳的說法，但**語言學證據不支持**。

- **唯一的古代文獻**：英國修士 **比德**（Bede，672-735 CE）在《論時間的計算》（*De Temporum Ratione*，725 CE）中記載：「Eosturmonath（復活月）的名字現在翻譯為『逾越月』，過去是以他們的一位女神 Eostre 命名的，人們在那個月慶祝她的節日。現在他們用她的名字來稱呼逾越季節。」
- **語言學追溯**：歷史語言學家將 Eostre 追溯到原始印歐語的黎明女神 \*H₂ewsōs，與希臘的 Eos（黎明女神）和羅馬的 Aurora 同源。這是日耳曼語族內部的詞源發展。
- **Ishtar 完全無關**：Ishtar 是美索不達米亞（閃族語系）的女神，與日耳曼語族（印歐語系）之間沒有語言學上的聯繫。Ishtar 的象徵是獅子和八角星，從未與兔子或蛋有關聯。兩個名字聽起來相似純粹是語音巧合（就像英文 "much" 和西班牙文 "mucho" 看起來相關但實際上不同源）。

結論：**Eostre 只貢獻了她的名字給英文和德文的這個節日名稱。** 世界上絕大多數語言稱這個節日為 Pascha（逾越節的變體）。Easter/Ishtar 的說法沒有學術基礎。

### 2. 兔子和彩蛋：不是第一世紀的，也不是聖經的

| 符號 | 最早可考的文獻 | 與第一世紀的關係 |
|------|---------------|----------------|
| **復活節兔子** | 1678 CE，Georg Franck von Franckenau 的 *De ovis paschalibus*（論復活節蛋），記載於德國西南部 | **無。** 第一世紀文獻中完全沒有兔子與逾越節或復活的關聯。 |
| **復活節彩蛋** | 中世紀歐洲（大齋期禁蛋 → 大齋期結束後裝飾蛋慶祝） | **無。** 第一世紀的逾越節食物是羔羊、無酵餅（מַצּוֹת / matzot）和苦菜（מְרֹרִים / merorim），見出埃及記 12:8。蛋出現在逾越節的 Seder 餐盤上（בֵּיצָה / beitzah）是後來拉比傳統的發展，時間不早於密西拿時期（~200 CE 編纂）。 |

一些網路文章聲稱兔子和蛋來自 Eostre 女神的「生育崇拜」。但牛津英語民俗詞典（*Oxford Dictionary of English Folklore*）指出：「現在很多作者聲稱野兔是盎格魯-撒克遜女神 Eostre 的聖獸，但沒有任何證據支持這一點；比德是唯一提到 Eostre 的作者，他沒有將她與任何動物聯繫起來。」

另外，一個在現代廣泛流傳的傳說——Eostre 把一隻冰凍的鳥變成了一隻會下蛋的兔子——**最早只能追溯到 1883 年**，沒有任何前基督教的民間傳說來源。

**出埃及記 12:8**（Sefaria）：
> וְאָכְל֥וּ אֶת־הַבָּשָׂ֖ר בַּלַּ֣יְלָה הַזֶּ֑ה צְלִי־אֵ֣שׁ וּמַצּ֔וֹת עַל־מְרֹרִ֖ים יֹאכְלֻֽהוּ
>
> RCUV：「當晚要吃羔羊的肉；要用火烤了，與無酵餅和苦菜一起吃。」

第一世紀逾越節的核心元素是：羔羊、無酵餅、苦菜、四杯酒、述說出埃及的故事（Haggadah 的早期形式）。兔子和彩蛋不在其中。

## 總結：經文與歷史的證據

| 問題 | 第一世紀的事實 | 後來的發展 |
|------|--------------|-----------|
| 復活與逾越節的關係 | 完全嵌入。耶穌在逾越節被釘，復活在初熟節。保羅稱基督為「我們的逾越節羔羊」和「初熟的果子」。 | 教會在 325 CE 尼西亞會議正式脫離猶太曆法 |
| 教會為何不在逾越節紀念 | 第一到第二世紀的信徒確實在逾越節紀念。Quartodeciman（十四日派）就是這個傳統的延續。 | 第二世紀羅馬教會開始推動星期日，第四世紀尼西亞會議定為正式規則 |
| Easter 來自 Ishtar？ | 不相關。Ishtar 是閃族女神，Easter 來自日耳曼語的 Eostre（黎明女神）。而且世界上大多數語言都叫這個節日 Pascha（逾越節）。 | 英文/德文採用了 Eostre 的月份名 |
| 兔子和彩蛋 | 第一世紀完全沒有相關證據。逾越節的食物是羔羊、無酵餅、苦菜。 | 兔子最早見於 1678 CE 德國；彩蛋來自中世紀大齋期傳統 |

這些是經文和歷史的證據。如何應用是你自己的決定。
