# 從第一世紀猶太觀點回應伊斯蘭教對聖經的三項宣稱

你的問題涉及三個獨立的宣稱，每一個都需要從希伯來文原文和第一世紀猶太文獻來檢驗。以下逐一分析。

---

## 一、篡改論（Tahrif）：聖經已被篡改，所以不可信？

### 歷史事實：抄本證據不支持篡改論

篡改論（تحريف）分為兩種形式：
- **tahrif bi-al-ma'ni**（意義篡改）——早期穆斯林學者如塔巴里（al-Tabari, 839–923 CE）的立場，認為猶太人和基督徒「誤讀」了經文
- **tahrif bi-al-lafz**（文本篡改）——伊本·哈茲姆（Ibn Hazm, 994–1064 CE）在11世紀發展出的立場，認為文本本身被竄改

從第一世紀猶太觀點來看，這兩種宣稱都有致命問題：

**死海古卷的證據**：1947年在昆蘭（Qumran）發現的死海古卷（約公元前3世紀至公元1世紀），包含幾乎所有希伯來聖經書卷的抄本。這些抄本與一千多年後的馬索拉文本（Masoretic Text）高度一致，僅存在極微小的拼寫差異。這證明希伯來聖經的文本從第二聖殿時期到中世紀基本未變。

**七十士譯本（LXX）的存在**：希臘文七十士譯本在公元前3世紀至前1世紀翻譯完成，提供了一個獨立於希伯來文傳統的文本見證。如果猶太人在某個時間點篡改了希伯來文本，七十士譯本會立刻暴露差異。

**新約抄本的規模**：新約現存約5,800份希臘文抄本、超過10,000份拉丁文抄本，以及數千份其他語言譯本，分佈在從英國到埃塞俄比亞、從西班牙到中國的廣大地區。在印刷術發明之前，要同時篡改散布在三大洲、數十種語言中的成千上萬份抄本，這在物理上是不可能的。

**第一世紀猶太人的經文觀**：約瑟夫（Josephus, 37–100 CE）在《反阿庇安》（*Contra Apionem* 1.42）中寫道，猶太人寧可為經文而死也不願更改一個字。第一世紀的猶太文士傳統（soferim）對抄寫有極嚴格的規範，包括逐字計數和校對機制。

> **結論**：篡改論是11世紀的伊斯蘭護教神學建構，不是基於文獻學證據的結論。死海古卷直接反駁了文本篡改的可能性。

## 二、申命記 18:18——「從他們弟兄中間」是指以實瑪利的後裔嗎？

### 經文全文

**希伯來原文**（來源：Sefaria）：

> **18:15** נָבִ֨יא מִקִּרְבְּךָ֤ מֵאַחֶ֙יךָ֙ כָּמֹ֔נִי יָקִ֥ים לְךָ֖ יְהֹוָ֣ה אֱלֹהֶ֑יךָ אֵלָ֖יו תִּשְׁמָעֽוּן׃
>
> **18:18** נָבִ֨יא אָקִ֥ים לָהֶ֛ם מִקֶּ֥רֶב אֲחֵיהֶ֖ם כָּמ֑וֹךָ וְנָתַתִּ֤י דְבָרַי֙ בְּפִ֔יו וְדִבֶּ֣ר אֲלֵיהֶ֔ם אֵ֖ת כׇּל־אֲשֶׁ֥ר אֲצַוֶּֽנּוּ׃

**和合本修訂版**（來源：Bible Gateway RCUV）：

> [15]「耶和華——你的 神要從你弟兄中給你興起一位先知像我，你們要聽他。」
>
> [18]「我必在他們弟兄中給他們興起一位先知像你。我要將當說的話放在他口裏；他要將我一切所吩咐的都告訴他們。」

**呂振中譯本**（來源：信望愛）：

> [15]「永恆主你的上帝要從你中間、從你的族弟兄中、給你興起一位神言人來、像我一樣——你們要聽他——」
>
> [18]「我要從他們的族弟兄中給他們興起一位神言人、像你一樣；我要把我的話放在他口中，他必將我所吩咐他的一切話都對他們說。」

### 希伯來文關鍵詞分析

核心詞組是 **מִקֶּ֥רֶב אֲחֵיהֶ֖ם**（miqerev acheihem），意為「從他們弟兄中間」。

伊斯蘭教的論點是：「弟兄」（אַחִים / achim）指以實瑪利的後裔（阿拉伯人），因為以實瑪利是以撒的兄弟，所以以實瑪利人是以色列人的「弟兄」。

**這個論點從希伯來文文脈來看完全站不住腳。原因如下：**

#### 1. 申命記的内部用法：「弟兄」= 以色列同胞

在整卷申命記中，אַח（ach，「弟兄」）一詞出現約20次以上，其中至少14次明確指「以色列同胞」。最關鍵的平行經文就在同一章和前一章：

**申命記 18:1-2**（來源：Sefaria + Bible Gateway RCUV）：

> 希伯來文：וְנַחֲלָ֥ה לֹא־יִֽהְיֶה־לּ֖וֹ **בְּקֶ֣רֶב אֶחָ֑יו**
>
> RCUV：「他在**弟兄中**沒有產業」

這裏說利未人在「弟兄中」沒有產業——「弟兄」顯然是指其他以色列支派，不是以實瑪利人。注意這裏用了完全相同的結構 **בְּקֶרֶב אֶחָיו**（beqerev echav），與18:18的 **מִקֶּרֶב אֲחֵיהֶם** 完全平行。

**申命記 17:14-15**（來源：Sefaria + Bible Gateway RCUV）：

> 希伯來文：**מִקֶּ֣רֶב אַחֶ֗יךָ** תָּשִׂ֤ים עָלֶ֙יךָ֙ מֶ֔לֶךְ לֹ֣א תוּכַ֗ל לָתֵ֤ת עָלֶ֙יךָ֙ **אִ֣ישׁ נׇכְרִ֔י** אֲשֶׁ֥ר **לֹֽא־אָחִ֖יךָ** הֽוּא
>
> RCUV：「要**從你弟兄中**立一人為你的王，不可立你弟兄之外的**外邦人**治理你。」

這段經文是決定性的。摩西使用了完全相同的語法結構 **מִקֶּרֶב אַחֶיךָ**（miqerev achecha，「從你弟兄中間」），並且明確對比：「弟兄」的反面是 **אִישׁ נׇכְרִי**（ish nokhri，「外邦人」）。按照伊斯蘭教的邏輯，如果「弟兄」包括以實瑪利後裔，那麼一個阿拉伯人就有資格做以色列的王——這顯然與經文的意圖矛盾。

#### 2. מִקִּרְבְּךָ 的用法鎖定了範圍

在18:15，摩西先說 **מִקִּרְבְּךָ**（miqirbecha，「從你中間」），然後才說 **מֵאַחֶיךָ**（me'achecha，「從你弟兄中」）。這兩個詞組是同位語（apposition），互相解釋：「從你中間」=「從你弟兄中」= 從以色列人中間。一個以實瑪利人不是「從你中間」。

#### 3. 聖經對以實瑪利的定位

**創世記 17:19-21**（來源：Sefaria）：

> וַיֹּ֣אמֶר אֱלֹהִ֗ים אֲבָל֙ שָׂרָ֣ה אִשְׁתְּךָ֗ יֹלֶ֤דֶת לְךָ֙ בֵּ֔ן ... וַהֲקִמֹתִ֨י אֶת־**בְּרִיתִ֥י** אִתּ֛וֹ לִבְרִ֥ית עוֹלָ֖ם לְזַרְע֥וֹ אַחֲרָֽיו׃ ... וְאֶת־**בְּרִיתִ֖י** אָקִ֣ים אֶת־**יִצְחָ֑ק**

神明確說：「我的約（בְּרִית / berit）要與以撒立」，不是與以實瑪利。以實瑪利蒙應許成為大國（גּוֹי גָּדוֹל），但約的傳承線是：亞伯拉罕→以撒→雅各→十二支派。申命記18的「弟兄」是在這條約的傳承線內部說的。

#### 4. 第一世紀的猶太理解

**4QTestimonia（4Q175）**——死海古卷昆蘭洞穴4出土，抄寫於公元前1世紀，是一份彌賽亞經文匯編。它引用了申命記18:18-19，將「像摩西的先知」理解為一位末世性的以色列先知。昆蘭社群（1QS 9:9-11）期待三位末世人物：一位先知、一位祭司彌賽亞、一位王者彌賽亞——三位都來自以色列。

**使徒行傳 3:22**（來源：Bible Gateway RCUV）：

> 摩西曾說：『主——你們的 神要**從你們弟兄中**給你們興起一位先知像我，凡他向你們所說的一切，你們都要聽從。』

彼得——一個第一世紀猶太人——引用申命記18:18時，直接將其應用在拿撒勒人耶穌身上（使徒行傳3:26），耶穌是猶大支派的以色列人。

**使徒行傳 7:37**（來源：Bible Gateway RCUV）：

> 這人是摩西，就是那曾對以色列人說「 神要**從你們弟兄中**給你們興起一位先知像我」的。

司提反——另一個第一世紀猶太人——同樣引用此經文，上下文是在講述以色列的歷史。

> **結論**：希伯來文 מִקֶּרֶב אֲחֵיהֶם 在申命記的一致用法中明確指「以色列同胞」。同一卷書17:15直接將「弟兄」定義為非外邦人。4QTestimonia和新約的第一世紀猶太作者都將此經文理解為指向以色列內部的先知。將「弟兄」解讀為以實瑪利後裔，與希伯來文的語法結構和整卷申命記的用法完全矛盾。

## 三、約翰福音 14:16 的 παράκλητος（parakletos）是 περίκλυτος（periklutos）= 穆罕默德嗎？

### 經文全文

**和合本修訂版**（來源：Bible Gateway RCUV）：

> [14:16] 我要求父，父就賜給你們**另外一位保惠師**，使他永遠與你們同在。
>
> [14:17] 他就是**真理的靈**，是世人不能接受的。因為他們既**看不見他**，也不認識他；你們卻認識他，因他常與你們同在，也要**在你們裏面**。
>
> [14:26] 但保惠師，就是父因我的名所要差來的**聖靈**，他要把一切的事教導你們，並且要使你們想起我對你們所說的一切話。

**新漢語譯本**（來源：CCV）：

> [14:16] 我要向父求，他就要賜給你們**另一位保惠師**，使他永遠與你們同在。

**呂振中譯本**（來源：信望愛）：

> [14:16] 我要求父，他就必將另一位**幫助者**賜給你們，叫他永遠和你們同在；

### 宣稱的核心：παράκλητος 被篡改自 περίκλυτος？

伊斯蘭教的論點是：
1. 約翰福音原文寫的是 **περίκλυτος**（periklutos，意為「被讚美者」）
2. 基督徒將其篡改為 **παράκλητος**（parakletos，「保惠師/幫助者」）
3. periklutos 的意思等同於阿拉伯文的 Ahmad（أحمد），即穆罕默德的另一個名字

**這個論點從希臘文抄本學和語言學來看完全沒有根據：**

#### 1. 零抄本支持

現存約5,800份希臘文新約抄本——從2世紀的蒲草紙殘片到中世紀的手抄本——**沒有任何一份**寫的是 περίκλυτος。全部都是 παράκλητος。這包括：
- **P66**（約125–175 CE）——最早的約翰福音抄本之一
- **P75**（約175–225 CE）
- **Codex Sinaiticus**（4世紀）
- **Codex Vaticanus**（4世紀）

如果早期基督徒篡改了這個詞，他們需要在短短幾十年內——在約翰福音流傳到羅馬、埃及、敘利亞各地之前——同時更改所有抄本。這在第一至第二世紀的抄本傳播條件下是不可能的。

#### 2. 兩個詞的詞根完全不同

| | παράκλητος (parakletos) | περίκλυτος (periklutos) |
|---|---|---|
| **詞根** | παρα- (para, 在旁邊) + καλέω (kaleo, 呼召) | περί- (peri, 周圍) + κλύω (klyo, 聽) |
| **意義** | 被呼召來身旁的人 → 辯護者、幫助者 | 被四方聽聞的人 → 著名的 |
| **詞性** | 動詞形容詞（法律用語） | 動詞形容詞（詩歌用語） |
| **用法** | 法庭辯護、猶太文獻中指代禱天使 | 荷馬史詩等古典希臘文學 |

這兩個詞的構詞法（morphology）、語意場（semantic field）、使用語境完全不同。它們在希臘文中不會被混淆，就像中文的「辯護」和「光榮」不會被混淆一樣。

#### 3. 經文自己定義了 parakletos 是什麼

約翰福音14-16章中，耶穌自己反覆描述這位 parakletos 的特徵：

| 經文 | 描述 | 穆罕默德是否符合？ |
|------|------|---------------------|
| 14:16「永遠與你們同在」 | 永久同在，不離開 | 穆罕默德是凡人，已去世 |
| 14:17「真理的靈」（τὸ πνεῦμα τῆς ἀληθείας） | 是一個「靈」（πνεῦμα / pneuma） | 穆罕默德是人，不是靈 |
| 14:17「世人看不見他」 | 不可見的存在 | 穆罕默德是可見的肉身之人 |
| 14:17「在你們裏面」（ἐν ὑμῖν） | 住在信徒裏面 | 穆罕默德不住在任何人裏面 |
| 14:26「就是……聖靈」（τὸ πνεῦμα τὸ ἅγιον） | 直接被稱為聖靈 | 穆罕默德不是聖靈 |
| 15:26「從父出來」（παρὰ τοῦ πατρὸς ἐκπορεύεται） | 從神那裏出來的 | 穆罕默德是人類母親所生 |
| 16:13「不是憑着自己說的」 | 傳達從神聽見的話 | — |
| 16:14「他要榮耀我」 | 這位 parakletos 的職責是榮耀耶穌 | 伊斯蘭教不以榮耀耶穌為核心 |

經文在14:26直接以同位語（apposition）定義：「保惠師，**就是**父因我的名所要差來的**聖靈**」（ὁ παράκλητος, τὸ πνεῦμα τὸ ἅγιον）。這不是後人的解釋——這是經文自己的定義。

#### 4. 第一世紀猶太背景中的 parakletos

在猶太文獻中，希伯來文/亞蘭文 **פרקליט**（peraklit）——直接從希臘文 parakletos 音譯而來——指的是在神的法庭上為人代求的辯護者、代禱者。密什拿（Mishnah, *Avot* 4:11）中有：「行了一條誡命的人，就為自己得了一位 peraklit（辯護者）」。

這個法律隱喻完全符合約翰福音的語境：耶穌自己被稱為第一位 parakletos（約翰一書 2:1），聖靈是「另一位」（ἄλλον / allon，「同類的另一位」）parakletos。這是在猶太法庭辯護的框架內說的。

#### 5. 這個宣稱的來源

periklutos = Ahmad 的論點最早出現在大衛·本雅明·凱爾達尼（David Benjamin Keldani, 1928）的著作中。正如學者 A. Guthrie 和 E. F. F. Bishop 指出的：「早期翻譯者對 periklutos 代替 parakletos 的猜測一無所知……伊本·伊斯哈格和伊本·希沙姆都沒有涉及 periklutos。」換言之，連最早的伊斯蘭歷史學家都不知道這個論點。

> **結論**：5,800份希臘文抄本一致讀作 παράκλητος，無一讀作 περίκλυτος。兩個詞的詞根、語意、用法完全不同。約翰福音14:26直接定義 parakletos =「聖靈」（τὸ πνεῦμα τὸ ἅγιον）——一個不可見的、住在信徒裏面的靈，不是六百年後出生的肉身之人。

## 綜合回應：從第一世紀猶太觀點看

作為一個第一世紀的妥拉學者（חכם），我的評估如下：

1. **篡改論**在文獻學上不成立。死海古卷（公元前3世紀至公元1世紀）證明希伯來聖經的文本穩定性。新約5,800份希臘文抄本的一致性使得系統性篡改在物理上不可能。這是11世紀伊斯蘭護教學的建構，不是基於證據的歷史結論。

2. **申命記18:18的「弟兄」**在希伯來文和申命記全書的一致用法中明確指以色列同胞。同一卷書17:15用完全相同的語法結構 מִקֶּרֶב אַחֶיךָ 並明確將「弟兄」與「外邦人」（נׇכְרִי）對立。4QTestimonia（公元前1世紀昆蘭抄本）和新約的第一世紀猶太作者一致將此經文理解為指向以色列內部的先知。

3. **parakletos = periklutos** 的宣稱沒有任何抄本依據（零份），兩個詞的希臘文詞根和語意完全不同，而且經文自己在約翰福音14:26以同位語直接定義 parakletos =「聖靈」——一個不可見的、住在信徒內部的靈。這個論點是20世紀的建構，連最早的伊斯蘭歷史學家都不知道。

**坦白說**：這三個宣稱都不是從文本內部產生的解讀，而是從外部的神學需要出發，然後回頭在文本中尋找支持。這正是本研經方法所反對的——讓文本說自己要說的話，而不是把外部議程讀進文本（eisegesis）。
