# 釐清方向

你提到了好幾個在第一世紀尚不存在的概念：

- **三位一體** (Trinity) — 三至四世紀大公會議的產物
- **位格** (hypostasis/persona) — 希臘哲學術語，後被教會借用
- **道成肉身** (Incarnation) — 尼西亞信經的神學建構
- **原罪** (Original Sin) — 奧古斯丁（四至五世紀）的教義
- **寶血代贖** (Penal Substitutionary Atonement) — 宗教改革時期的理論

這些概念都是後來教會發展出來的神學框架，不是約翰福音的作者或耶穌自己使用的語言。

你想了解什麼？

- **第一世紀的原始概念** — 了解約翰福音 3:16 在教會發展這些概念之前的原始意涵
- **概念發展史** — 追溯這些教義從第一世紀到後期教會的發展過程
- **希伯來文原意** — 回到希伯來原文，看原始用詞的語意範圍

---

由於你的問題包含了具體的經文（約翰福音 3:16）和明確的神學主張，以下我先提供第一世紀的原始概念解讀，讓你看到經文在這些教義被發明之前，究竟在說什麼。

## 約翰福音 3:14-21 — 第一世紀猶太語境下的解讀

### 第一世紀語境：這段經文在說什麼

#### 1. 「舉蛇」的典故 — 銅蛇事件（民數記 21:4-9）

約翰福音 3:14 直接引用了妥拉的敘事。不理解這個典故，就無法理解 3:16。

**民數記 21:8-9 希伯來原文** (來源: Sefaria):
> וַיֹּ֨אמֶר יְהֹוָ֜ה אֶל־מֹשֶׁ֗ה עֲשֵׂ֤ה לְךָ֙ שָׂרָ֔ף וְשִׂ֥ים אֹת֖וֹ עַל־נֵ֑ס וְהָיָה֙ כׇּל־הַנָּשׁ֔וּךְ וְרָאָ֥ה אֹת֖וֹ וָחָֽי׃

**和合本修訂版**:
> [8] 耶和華對摩西說：「你要造一條火蛇，掛在杆子上。凡被咬的，一望這蛇就必存活。」

注意這個結構：**被蛇咬的人 → 仰望銅蛇 → 存活**。這不是「代替」的邏輯，而是「仰望/信靠」的邏輯。銅蛇沒有「代替」任何人被咬，也沒有「承受」蛇毒。被蛇咬的人透過「仰望」（רָאָה / ra'ah）而得到醫治。

約翰福音的作者用這個典故來解釋「人子被舉起來」的意義：重點在於「仰望/相信」帶來的生命，不在於「代替受罰」。

#### 2. 「神愛世人」— 不是三位一體的內部運作

希臘原文用的是 ἠγάπησεν (egapesen) —「愛」的過去式。這個動詞在第一世紀猶太語境中，對應的是希伯來概念 אהבה (ahavah) 和更重要的 חֶסֶד (hesed)。

חֶסֶד (hesed) 是妥拉中描述 YHWH 與以色列關係的核心詞彙 — 它不是抽象的情感，而是**盟約內的忠誠委身** (covenant loyalty)。當約翰福音說「神愛世人」，在第一世紀猶太聽眾耳中，這是在說 YHWH 將祂的盟約忠誠擴展到整個世界 — 這是一個關於以色列的神對萬國的 hesed 的宣告。

**這裡完全沒有「三位一體的第二位格」的概念。** 三位一體教義是三至四世紀大公會議的產物：

- 尼西亞會議 (325 CE) 首次正式定義「聖子與聖父同質」(ὁμοούσιος)
- 君士坦丁堡會議 (381 CE) 確立三位一體的完整教義
- 迦克墩會議 (451 CE) 定義「二性一位格」

**第一世紀的猶太人每天誦讀的是示瑪 (Shema)：**

**申命記 6:4 希伯來原文** (來源: Sefaria):
> שְׁמַ֖ע יִשְׂרָאֵ֑ל יְהֹוָ֥ה אֱלֹהֵ֖ינוּ יְהֹוָ֥ה אֶחָֽד

「聽啊，以色列！YHWH 我們的神，YHWH 是 **אֶחָד** (echad / 一)。」

אֶחָד 的意思是「一」。耶穌自己在馬可福音 12:29 被問到最大的誡命時，回答的正是示瑪。一個天天誦讀示瑪的加利利猶太拉比，不可能用「三位一體的第二位格」來理解自己。

#### 3. 「獨一的兒子」— μονογενής 的語意

希臘文 μονογενής (monogenes) 在中文被譯為「獨生子」或「獨一的兒子」。KJV 譯為 "only begotten Son"。

但 μονογενής 的語意不是「被生出來的」(begotten)，而是「獨一無二的」(unique, one-of-a-kind)。這個詞在七十士譯本 (LXX) 中用來翻譯希伯來文 יָחִיד (yachid)，比如用在亞伯拉罕獻以撒的敘事中（創世記 22:2 — 「你獨一的兒子」）。

在第一世紀猶太語境中，「神的兒子」(בֶּן אֱלֹהִים / ben Elohim) 不是指「本體上與神同質的第二位格」。在希伯來聖經中，「神的兒子」可以指：

- 以色列全民族（出埃及記 4:22 — 「以色列是我的兒子，我的長子」）
- 以色列的王（詩篇 2:7 — 「你是我的兒子，我今日生你」）
- 天上的存有（約伯記 1:6 — בְּנֵי הָאֱלֹהִים）
- 義人（智慧書 2:18）

約翰福音的作者將耶穌描述為神「獨一的」(μονογενής) 被差遣者，是在彌賽亞的框架內說話 — 神差遣祂所揀選的、獨特的使者來完成救贖使命 — 而不是在描述三位一體內部的本體論關係。

#### 4. 「原罪」— 第一世紀不存在的概念

你說「原罪讓人無法自救」。這個教義來自奧古斯丁 (354-430 CE)，不是來自聖經。

第一世紀猶太人的概念是 **יֵצֶר** (yetzer / 傾向)：

**創世記 6:5 希伯來原文** (來源: Sefaria):
> וְכׇל־יֵ֙צֶר֙ מַחְשְׁבֹ֣ת לִבּ֔וֹ רַ֥ק רַ֖ע כׇּל־הַיּֽוֹם

「他心中思想的一切 **יֵצֶר** (傾向)，終日都是惡的。」

**創世記 8:21 希伯來原文** (來源: Sefaria):
> כִּ֠י יֵ֣צֶר לֵ֧ב הָאָדָ֛ם רַ֖ע מִנְּעֻרָ֑יו

「因為人心的 **יֵצֶר** (傾向) 從年幼就是惡的。」

猶太傳統認為人有 **יֵצֶר הָרָע** (yetzer ha-ra / 惡的傾向) 和 **יֵצֶר הַטּוֹב** (yetzer ha-tov / 善的傾向)。這不是「遺傳的罪性使人完全無法自救」。恰恰相反，妥拉明確教導人有能力選擇：

**創世記 4:7 希伯來原文** (來源: Sefaria):
> הֲל֤וֹא אִם־תֵּיטִיב֙ שְׂאֵ֔ת וְאִם֙ לֹ֣א תֵיטִ֔יב לַפֶּ֖תַח חַטָּ֣את רֹבֵ֑ץ וְאֵלֶ֙יךָ֙ תְּשׁ֣וּקָת֔וֹ **וְאַתָּ֖ה תִּמְשׇׁל־בּֽוֹ**

YHWH 對該隱說：「你若做得好，豈不仰起頭來嗎？你若做得不好，罪就伏在門口。它想要控制你，但**你可以制伏它**。」

וְאַתָּה תִּמְשׇׁל־בּוֹ — 「而你，你能治理它。」這直接否定了「人完全無法自救」的前提。

以西結書更明確地否定了「罪的遺傳」：

**以西結書 18:20** (來源: Sefaria / Bible Gateway RCUV):
> הַנֶּ֥פֶשׁ הַחֹטֵ֖את הִ֣יא תָמ֑וּת בֵּ֞ן לֹא־יִשָּׂ֣א בַּעֲוֺ֣ן הָאָ֗ב וְאָב֙ לֹ֤א יִשָּׂא֙ בַּעֲוֺ֣ן הַבֵּ֔ן

「惟有犯罪的，卻必死亡。**兒子不擔當父親的罪孽，父親也不擔當兒子的罪孽。**」

如果兒子不擔當父親的罪孽，那「亞當的罪遺傳給全人類」的教義就直接與以西結書衝突。第一世紀猶太人不會這樣讀經。

#### 5. 「寶血代贖」— 妥拉中的贖罪機制不是「刑罰替代」

你說的「寶血代贖」反映的是 **刑罰替代論** (Penal Substitutionary Atonement)，這是宗教改革時期（十六世紀）加爾文發展的理論，不是第一世紀的概念。

妥拉中的贖罪機制是聖殿獻祭系統。關鍵經文：

**利未記 17:11 希伯來原文** (來源: Sefaria):
> כִּ֣י נֶ֣פֶשׁ הַבָּשָׂר֮ בַּדָּ֣ם הִוא֒ וַאֲנִ֞י נְתַתִּ֤יו לָכֶם֙ עַל־הַמִּזְבֵּ֔חַ **לְכַפֵּ֖ר** עַל־נַפְשֹׁתֵיכֶ֑ם כִּֽי־הַדָּ֥ם ה֖וּא בַּנֶּ֥פֶשׁ **יְכַפֵּֽר**

關鍵詞是 **כָּפַר** (kafar)。這個希伯來字根的語意範圍是「遮蓋、塗抹、清除」，與 Yom Kippur（יוֹם כִּפּוּר / 贖罪日）的 כִּפּוּר 同根。

כָּפַר 在妥拉中的功能是**恢復盟約關係、潔淨不潔、消除過犯** — 不是「無辜者代替有罪者承受神的刑罰」。聖殿獻祭中：

- 贖罪祭 (חַטָּאת / chatat) 的功能是**潔淨** (purification)，不是「代替受罰」
- 贖愆祭 (אָשָׁם / asham) 的功能是**賠償/修復**，不是「代替受罰」
- Yom Kippur 的替罪羊 (עֲזָאזֵל / azazel) 是被放逐到曠野，不是被殺死 — 它「帶走」(נָשָׂא / nasa) 罪孽，不是「代替受刑」

此外，妥拉的獻祭系統處理的主要是無意之罪 (שְׁגָגָה / shgagah) 和禮儀上的不潔 (טֻמְאָה / tumah)。故意犯罪 (בְּיָד רָמָה / b'yad ramah / 「高舉的手」) 需要的是 **תְּשׁוּבָה** (teshuvah / 悔改 — 字面意思是「轉回」)，不是血祭。

#### 6. 那約翰福音 3:16 在第一世紀到底在說什麼？

讀回完整上下文（3:14-21），這段話的核心結構是：

1. **銅蛇的類比** (v.14-15)：就像以色列人在曠野仰望被舉起的銅蛇而存活，人仰望被舉起的人子而得到永恆的生命。
2. **YHWH 的盟約忠誠擴展** (v.16-17)：神的 hesed 不限於以色列，而是面向全世界 (κόσμος / kosmos)。神差遣祂獨一的使者，不是來定罪 (κρίνω / krino)，而是來帶來拯救 (σῴζω / sozo)。
3. **光與暗的二元** (v.19-21)：這個意象在死海古卷（特別是社群規章 1QS 3:13-4:26）中有清楚的平行 — 光明之子與黑暗之子的對立。行真理的人靠近光，作惡的人逃避光。

**關鍵觀察：第 20-21 節強調的是「行為」（ἔργα / erga），不是「信條」。**

> v.20：凡**作惡的人**都恨惡光
> v.21：但**實行真理的人**就來接近光，為要顯明他的**行為**是靠神而行的。

這段經文的結尾明確地把焦點放在人的行為上。在第一世紀猶太框架中，「信」(πιστεύω / pisteuo) 不是指「頭腦中接受一套教義」，而是 **אמונה** (emunah / 信實、忠誠) — 一種以行動表現出來的對 YHWH 的忠誠委身。

### 常見誤讀

| 後來的教義詮釋 | 第一世紀的原始意涵 |
|---|---|
| 三位一體的第二位格道成肉身 | YHWH 差遣祂獨一的彌賽亞使者（示瑪：YHWH 是一） |
| 原罪使人完全無法自救 | יֵצֶר (yetzer / 傾向) — 人有惡的傾向但可以制伏（創 4:7） |
| 耶穌的血代替人承受神的刑罰 | כָּפַר (kafar) — 潔淨、恢復盟約關係，不是刑罰替代 |
| 「信」= 接受教義命題 | אמונה (emunah) — 以行動表現的忠誠委身 |
| 3:16 是核心教義的摘要 | 3:14-21 是一個完整的論述，以銅蛇類比開始，以行為的審判結束 |

### 翻譯偏差提示

- **「獨生子」** — μονογενής 的意思是「獨一的」(unique)，不是「被生出來的」(begotten)。KJV 的 "only begotten" 受到後期三位一體神學影響。RCUV 修訂為「獨一的兒子」較為準確。
- **「得救」** — 希臘文 σῴζω (sozo) 的語意範圍包括「拯救、醫治、保全」，在猶太語境中對應 יָשַׁע (yasha) — 從具體危難中被拯救出來，不是指「靈魂永恆的得救」。
- **「定罪」** — 希臘文 κρίνω (krino) 更準確的翻譯是「審判」，不一定是定罪。呂振中譯本的處理較好。
