# 「宗教都一樣」？從第一世紀猶太視角檢視三個慈濟式比較

你的問題涉及三個常見的宗教融合論述（syncretic claims），在台灣尤其普遍。以下逐一從第一世紀希伯來文和猶太脈絡進行分析。

---

## 一、「你們願意人怎樣待你們…」＝佛教慈悲喜捨？

### 經文

**馬太福音 7:12**（來源：Bible Gateway RCUV）：

> 所以，無論何事，你們想要人怎樣待你們，你們也要怎樣待人，因為**這就是律法和先知的道理**。

呂振中譯本：

> 所以無論如何，你們願意人怎樣待你們，你們也要怎樣待人：**這就是律法和神言人道理的總綱**。

新漢語譯本：

> 所以，無論甚麼事，你們願意人怎樣對待你們，你們就要怎樣對待人，因為**這就是律法和先知的教導**。

### 上下文（馬太福音 7:7-14，來源：Bible Gateway RCUV）

> [7] 你們祈求，就給你們；尋找，就找到；叩門，就給你們開門。[8] 因為凡祈求的，就得着；尋找的，就找到；叩門的，就給他開門。[9] 你們中間誰有兒子求餅，反給他石頭呢？[10] 求魚，反給他蛇呢？[11] 你們雖然不好，尚且知道拿好東西給兒女，**何況你們在天上的父，他豈不更要把好東西賜給求他的人嗎？** [12] 所以，無論何事，你們想要人怎樣待你們，你們也要怎樣待人，因為這就是律法和先知的道理。[13] 你們**要進窄門**。因為通往滅亡的門是寬的，路是大的，進去的人也多；[14] 通往生命的門是窄的，路是小的，找到的人也少。

### 第一世紀脈絡

這句話不是一條獨立的道德格言。Yeshua（耶穌）的原話結構是一個 **kal va-chomer（קל וחומר，「從輕到重」）** 推論——這是第一世紀拉比辯論的標準邏輯方法：

1. **輕**（kal）：你們不好的人尚且知道給兒女好東西（7:11）
2. **重**（chomer）：**在天上的父**更要把好東西給求他的人
3. **結論**（7:12）：**所以**（希臘文 οὖν / oun），你們也要這樣待人

關鍵三個字：**「律法和先知」**（希臘文 ὁ νόμος καὶ οἱ προφῆται）。這不是泛泛的道德總結，而是一個具體的 Torah 宣告。Yeshua 說的是：**我現在教你們的，就是 Torah 和 Nevi'im 一直在教的。**

同一位 Yeshua 在馬太福音 22:36-40 被問到律法最大的誡命時，他引用的是：

**申命記 6:4-5**（Shema，來源：Sefaria）：

> שְׁמַע יִשְׂרָאֵל יְהֹוָה אֱלֹהֵינוּ יְהֹוָה אֶחָד׃ וְאָהַבְתָּ אֵת יְהֹוָה אֱלֹהֶיךָ בְּכׇל־לְבָבְךָ וּבְכׇל־נַפְשְׁךָ וּבְכׇל־מְאֹדֶךָ׃
>
> 以色列啊，你要聽！耶和華——我們的神是獨一的主。你要盡心、盡性、盡力愛耶和華——你的神。

**利未記 19:18**（來源：Sefaria）：

> וְאָהַבְתָּ לְרֵעֲךָ כָּמוֹךָ אֲנִי יְהֹוָה׃
>
> 你要愛鄰如己。**我是耶和華。**（來源：Bible Gateway RCUV）

注意利未記 19:18 的最後三個字：**אֲנִי יְהֹוָה**（ani YHWH，「我是耶和華」）。這不是一條通用的道德準則，而是 YHWH 以盟約主（covenant Lord）的身分發出的命令。「愛鄰如己」的權柄來源是 **YHWH 自己**——不是「宇宙法則」，不是「因果」，不是任何抽象的善。

### 常見誤讀

**誤讀**：「Golden Rule 是普世道德，所有宗教都有，所以宗教本質相同。」

**問題**：這種比較抽離了 Yeshua 這句話的整個框架。

| 面向 | Yeshua 的 Golden Rule（馬太福音 7:12） | 佛教的慈悲喜捨（四無量心） |
|------|---------------------------------------|--------------------------|
| **權柄來源** | Torah 和先知——YHWH 的盟約命令 | 佛陀的覺悟、法的體悟 |
| **動機結構** | 天父的恩賜（7:11）→ 因此你也如此待人 | 修行者的內在修為，斷除煩惱 |
| **對象** | 盟約群體中的「鄰人」（רֵעַ / rea），最終擴展到寄居者 | 一切有情眾生（無差別的普遍慈悲） |
| **目標** | 遵行 Torah、活出盟約關係 | 解脫輪迴、證菩提 |
| **緊接的下文** | 「要進**窄門**」——不是所有路都通（7:13-14） | 四無量心是八萬四千法門之一 |

**特別注意馬太福音 7:13-14 的窄門宣告**。Yeshua 在講完 Golden Rule 之後，**立刻**說「通往生命的門是窄的，路是小的，找到的人也少。」這直接否定了「條條大路通羅馬」式的宗教融合觀。在第一世紀猶太語境中，Yeshua 是在說：**只有 Torah 的路是通往生命的路。**

## 二、「人人都有佛性」＝「按著神的形像造人」（imago Dei）？

### 經文

**創世記 1:26-27**（來源：Sefaria 希伯來文 + Bible Gateway RCUV）：

> וַיֹּאמֶר אֱלֹהִים נַעֲשֶׂה אָדָם **בְּצַלְמֵנוּ** **כִּדְמוּתֵנוּ** וְיִרְדּוּ בִדְגַת הַיָּם…
>
> 神說：「我們要照着我們的**形像**（tselem），按着我們的**樣式**（demut）造人，使他們管理海裏的魚、天空的鳥…」

> וַיִּבְרָא אֱלֹהִים אֶת־הָאָדָם **בְּצַלְמוֹ** בְּצֶלֶם אֱלֹהִים בָּרָא אֹתוֹ זָכָר וּנְקֵבָה בָּרָא אֹתָם׃
>
> 神就照着他的形像創造人，照着神的形像創造他們；他創造了他們，有男有女。

新漢語譯本：

> 於是神照他的形像創造了人，照神的形像創造了他們——男和女。神創造了他們。

呂振中譯本：

> 上帝就按自己的形像創造人：按上帝的形像創造他：創造他們有男有女。

### 希伯來文分析

兩個關鍵詞：

- **צֶלֶם**（tselem）——「形像」。這個字在古近東語境中指的是**國王在遠方省份豎立的雕像**，代表國王的權柄和統治。亞述和巴比倫的銘文中（主前 9-7 世紀），tselem 用於皇家代表像（royal representative image）。在創世記的語境中，人類被造為神在地上的 **代理統治者**（vice-regent），這就是為什麼下一句立刻是「**使他們管理**（וְיִרְדּוּ / v'yirdu）」。

- **דְּמוּת**（demut）——「樣式」。不是「等同」，而是「相似、類比」。人像神，但人不是神。

### 第一世紀脈絡

在第一世紀猶太理解中，**tselem Elohim**（神的形像）意味著：

1. **功能性的委託**——人類受命代表神管理受造界（創世記 1:28）
2. **關係性的地位**——人類與神之間有獨特的盟約關係
3. **不可侵犯的尊嚴**——創世記 9:6 明確以 tselem 為基礎禁止謀殺

這與佛性（buddhadhatu / 佛性）的概念有根本性的區別：

| 面向 | צֶלֶם אֱלֹהִים（tselem Elohim） | 佛性 |
|------|----------------------------------|------|
| **本體論** | 受造物的被賦予地位——神給的，不是人本有的 | 一切眾生本具的覺性——自性本然 |
| **方向** | **由外而內**：神創造、神賦予、神委託 | **由內而外**：內在本具、修行開顯 |
| **造物主/受造物區分** | 絕對的。人永遠不會「成為神」。 | 界線是流動的。一切眾生皆可成佛。 |
| **倫理基礎** | 因為你是神的形像代表，所以你有責任（管理、不可殺人等） | 因為你有佛性，所以你有潛力覺悟 |
| **與「神性」的關係** | 人反映神的屬性，但不共享神的本質 | 眾生與佛共享同一本質（法性） |

### 常見誤讀

**誤讀**：「tselem Elohim 就是說人裡面有神性——所以跟佛教說人裡面有佛性是一樣的。」

**問題**：tselem 是**代理人的功能性標記**，不是**內在神性的本體論宣告**。

一個第一世紀猶太人聽到「人裡面有神性」會極度震驚。利未記 19:2 的命令是：

> קְדֹשִׁים תִּהְיוּ כִּי קָדוֹשׁ אֲנִי יְהֹוָה אֱלֹהֵיכֶם׃
>
> 你們要**成為聖**，因為我耶和華——你們的神**是神聖的**。（來源：Sefaria + Bible Gateway RCUV）

「成為聖」（קְדֹשִׁים תִּהְיוּ / qedoshim tihyu）是一個**命令式**——你要成為你還不是的狀態。這不是「你本來就是聖的，只是還沒發現」。這是：**YHWH 是聖的，你們要效法他。** 創造者與被造物之間的區分，在希伯來聖經中從未被模糊。

以賽亞書 43:10-12（來源：Sefaria + Bible Gateway RCUV）更加直接：

> אַתֶּם עֵדַי נְאֻם־יְהֹוָה... כִּי־אֲנִי הוּא לְפָנַי לֹא־נוֹצַר אֵל וְאַחֲרַי לֹא יִהְיֶה׃
>
> 你們是我的見證——耶和華說——…明白我就是耶和華。**在我以前沒有任何被造的真神，在我以後也必沒有。** …**除我以外沒有救主。**

在這個框架裡，任何一個受造物「本具神性」的說法都違反了希伯來聖經最基本的神學結構。

## 三、耶穌的捨命＝菩薩道的捨身？

### 經文

**馬可福音 10:45**（來源：Bible Gateway RCUV）：

> 因為人子來，並不是要受人的服事，乃是要服事人，並且要**捨命作多人的贖價**（λύτρον / lutron）。

**約翰福音 10:17-18**（來源：Bible Gateway RCUV）：

> 為此，我父愛我，因為我把命捨去，好再取回來。**沒有人奪去我的命，是我自己捨的；我有權捨棄，也有權再取回。** 這是我從我父所受的命令。

**以賽亞書 53:5-6, 10-12**（來源：Sefaria 希伯來文 + Bible Gateway RCUV）：

> וְהוּא מְחֹלָל מִפְּשָׁעֵנוּ מְדֻכָּא מֵעֲוֺנֹתֵנוּ מוּסַר שְׁלוֹמֵנוּ עָלָיו וּבַחֲבֻרָתוֹ נִרְפָּא־לָנוּ׃
>
> [5] 他為我們的過犯受害，為我們的罪孽被壓傷。因他受的懲罰，我們得平安；因他受的鞭傷，我們得醫治。
>
> [6] 我們都如羊走迷，各人偏行己路；**耶和華使我們眾人的罪孽都歸在他身上**。

> [10] 耶和華的旨意要壓傷他，使他受苦。**當他的生命作為贖罪祭（אָשָׁם / asham）時**，他必看見後裔…
>
> [12] …因為他**傾倒自己的生命（הֶעֱרָה לַמָּוֶת נַפְשׁוֹ），以至於死**，也列在罪犯之中。他卻**擔當多人的罪**，為他們的過犯**代求（יַפְגִּיעַ / yafgia）**。

**利未記 17:11**（來源：Sefaria + Bible Gateway RCUV）：

> כִּי נֶפֶשׁ הַבָּשָׂר בַּדָּם הִוא וַאֲנִי נְתַתִּיו לָכֶם עַל־הַמִּזְבֵּחַ **לְכַפֵּר** עַל־נַפְשֹׁתֵיכֶם כִּי־הַדָּם הוּא בַּנֶּפֶשׁ **יְכַפֵּר**׃
>
> 因為動物的生命是在血中。我把這血賜給你們，可以在祭壇上為你們的生命**贖罪**（kipper）；因為血就是生命，能夠**贖罪**。

### 第一世紀脈絡

Yeshua 的死，在第一世紀猶太框架中，有三個完全不同於菩薩道的結構要素：

#### 1. 贖價（λύτρον / lutron）——盟約法律概念

馬可福音 10:45 用的是 **λύτρον**（lutron），對應希伯來文 **כֹּפֶר**（kofer）——這是一個法律術語，指支付贖金以釋放被扣押的人（出埃及記 21:30；民數記 35:31-32）。這不是「犧牲精神」的道德示範，而是一個**具體的法律交易**：有人欠了債，有人代付了。

#### 2. 贖罪祭（אָשָׁם / asham）——聖殿獻祭系統

以賽亞書 53:10 用的是 **אָשָׁם**（asham），「贖罪祭」——這是利未記 5-7 章規定的五種祭之一，專門用於補償因罪造成的損害。這個字把「受苦的僕人」的死放進了**聖殿獻祭系統**的框架中。這不是泛泛的「為眾生犧牲」，而是：**YHWH 指定的獻祭方式，由 YHWH 發起，按照 Torah 的規定執行。**

利未記 17:11 明確說：**是 YHWH 把血賜給以色列人，作為贖罪的方式**（וַאֲנִי נְתַתִּיו לָכֶם / va'ani n'tativ lakhem，「我把它賜給你們」）。贖罪的權柄和方式完全由 YHWH 決定。

#### 3. 約翰福音 10:18 的獨特宣告——權柄而非無奈

「**沒有人奪去我的命，是我自己捨的；我有權捨棄，也有權再取回。**」

這不是被動的犧牲精神，而是一個關於**神聖權柄**的宣告。在第一世紀猶太語境中，宣稱自己有權柄「取回」生命——即從死裡復活——是一個極度激進的宣告，因為在希伯來聖經中，只有 YHWH 有使人從死裡復活的能力（申命記 32:39；撒母耳記上 2:6）。

### 與菩薩道的根本差異

| 面向 | Yeshua 的捨命 | 菩薩道的捨身 |
|------|--------------|-------------|
| **發起者** | YHWH 的旨意（賽 53:10「耶和華的旨意要壓傷他」） | 菩薩自己的悲願（本願力） |
| **機制** | Torah 獻祭系統——asham、kipper、kofer | 功德迴向、波羅蜜（paramita）的累積 |
| **對象** | 盟約百姓的罪（「為我百姓的罪過」，賽 53:8） | 一切有情眾生（無差別） |
| **目的** | 罪的赦免與盟約的恢復 | 眾生脫離苦海、證悟成佛 |
| **一次性 vs 反覆** | 一次性的、不可重複的事件（歷史中的特定時刻） | 菩薩多生累劫反覆捨身行道 |
| **復活** | Yeshua 宣稱有權取回生命（約 10:18） | 菩薩在輪迴中轉世再來 |
| **世界觀** | 線性歷史：創造→墮落→救贖→恢復 | 循環輪迴：無始以來的生死流轉 |

### 常見誤讀

**誤讀**：「都是犧牲自己來救眾生，本質是一樣的。」

**問題**：表面相似（兩者都涉及「為他人受苦」）掩蓋了完全不同的機制、目的和世界觀。

Yeshua 的死不是一個**道德示範**（「看，犧牲精神多偉大」），而是一個**盟約行動**——在 Torah 獻祭系統的框架中，由 YHWH 發起，按照 YHWH 的旨意執行，目的是處理**罪**（חֵטְא / chet，偏離 YHWH 的道路）和恢復**盟約關係**。

菩薩道的捨身則是在一個完全不同的形而上學框架中運作的：沒有 YHWH，沒有盟約，沒有 Torah 獻祭系統，沒有線性歷史觀，也沒有「罪」的概念（佛教用的是 akusala/不善，指引發苦的行為，不是對盟約神的冒犯）。

## 總結：為什麼「宗教都一樣」在第一世紀猶太視角中站不住腳

證嚴法師的「宗教都是一樣的，都是教人為善」這個命題，要成立的前提是：**宗教的核心是道德教導**。但在第一世紀猶太視角中，Torah 的核心不是「教人為善」，而是：

**出埃及記 20:2-3**（來源：Sefaria + Bible Gateway RCUV）：

> אָנֹכִי יְהֹוָה אֱלֹהֶיךָ אֲשֶׁר הוֹצֵאתִיךָ מֵאֶרֶץ מִצְרַיִם מִבֵּית עֲבָדִים׃ לֹא־יִהְיֶה־לְךָ אֱלֹהִים אֲחֵרִים עַל־פָּנָי׃
>
> 我是耶和華——你的神，曾將你從埃及地為奴之家領出來。**除了我以外，你不可有別的神。**

十誡的第一條不是「要做好事」，而是「**我是 YHWH，除我以外不可有別的神**。」Torah 的基礎是**排他性的盟約忠誠**（exclusive covenant loyalty）——這與「信什麼教都可以」不只是不同，而是直接矛盾。

Yeshua 自己在馬太福音 5:17-19 說：

> 不要以為我來是要廢掉**律法和先知**。我來不是要廢掉，而是要成全。…就是到天地都廢去，律法的**一點一畫**也不能廢去，直到一切都實現。

一個第一世紀的 Torah 教師，不會認為「所有宗教都一樣」，因為 Torah 本身——他視為永恆不變的文本——明確禁止這種想法。
