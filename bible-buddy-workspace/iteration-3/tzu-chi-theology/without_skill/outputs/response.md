# 從第一世紀希伯來/猶太脈絡檢視慈濟神學與聖經的類比

lancetw，這是一個非常好的問題，涉及多層次的跨宗教比較。以下我會從第一世紀的希伯來文本和猶太思想脈絡逐一分析這三組類比。

---

## 一、「己所欲，施於人」vs「慈悲喜捨」

### 經文：馬太福音 7:12

> 「所以，無論何事，你們願意人怎樣待你們，你們也要怎樣待人，因為這就是律法和先知的道理。」

### 猶太脈絡分析

這句話在猶太傳統中並非獨立的倫理格言，而是對整部妥拉（Torah）的總結。耶穌自己說得很清楚：「因為這就是律法和先知的道理」（houtos gar estin ho nomos kai hoi prophetai）。在希伯來思維中，「律法和先知」指的是整個立約關係的框架——神與以色列之間的盟約（berith, בְּרִית）。

拉比希列（Hillel）在約公元前 20 年也有類似的表述：「你所憎惡的，不要施於你的同伴。這就是整部妥拉，其餘都是註解。」（《巴比倫塔木德》安息日篇 31a）。但關鍵是：希列同樣將此話錨定在妥拉——即盟約——之上。

**核心差異**：在猶太-基督教脈絡中，倫理行為的根基是「盟約關係」——人之所以要善待他人，是因為神先立約、先施恩。倫理是「回應性」的（responsive），而非自發性的。佛教的「慈悲喜捨」（四無量心：metta, karuna, mudita, upekkha）則源自對一切有情眾生之苦的覺察，其動力來自對「苦」（dukkha）的洞見和對「無我」（anatta）的實踐。

換句話說：

- **聖經倫理的邏輯**：神先愛了我們 --> 因此我們當彼此相愛（約翰一書 4:19）。倫理有一個位格性的源頭。
- **佛教倫理的邏輯**：一切有情皆苦 --> 因此生起慈悲。倫理來自對實相的覺悟。

表面上的行為可能相似（善待他人），但驅動邏輯、形上學基礎完全不同。

## 二、「人人都有佛性」vs「按著神的形像造人」

### 經文：創世記 1:27

> 「神就照著自己的形像造人，乃是照著神的形像造他；造男造女。」

### 希伯來文分析

「神的形像」（tselem Elohim, צֶלֶם אֱלֹהִים）在古代近東的語境中有非常具體的含義。在美索不達米亞和埃及的王室意識形態中，「神的形像」（tselem）是指國王作為神在地上的「代表」或「代理人」（representative）。創世記的革命性宣告在於：不只是法老或國王，而是「每一個人」都是神的形像。

但「tselem」的意思是「代表」，而非「內在本質」。人是神的形像，不是因為人的內在有一個「神性」，而是因為人被賦予了一個「功能」和「使命」——在受造界中代表神治理大地（創世記 1:28）。

**「佛性」的概念**根本不同。佛性（buddha-dhatu / tathagata-garbha）指的是一切有情眾生內在本具的覺悟潛能，是一種本體論的主張——每個人的本質中「已經有」佛的種子。

**核心差異**：

| | 神的形像（tselem Elohim） | 佛性（buddha-dhatu） |
|---|---|---|
| **本質** | 被賦予的「功能/身份」 | 內在本具的「本質」 |
| **方向** | 由外而內（神賦予人） | 由內而外（本自具足） |
| **關係** | 造物主與受造者之間的質的差別始終存在 | 眾生與佛之間的差別最終可消除 |
| **目標** | 人永遠不會「變成」神；人是代表神的管家 | 眾生可以成佛（覺悟即是佛） |

在第一世紀的猶太思想中，神與人之間的界線（Creator-creature distinction）是絕對不可跨越的。任何宣稱人的內在擁有「神性」的主張，對第一世紀的猶太人而言會接近褻瀆。

## 三、耶穌的捨命 vs 菩薩道的捨身

### 經文脈絡

耶穌在馬可福音 10:45 說：「因為人子來，並不是要受人的服事，乃是要服事人，並且要捨命，作多人的贖價（lytron anti pollon）。」

### 第一世紀猶太脈絡分析

「贖價」（lytron, 希伯來文 kofer / כֹּפֶר）在舊約中是一個明確的盟約和獻祭用語。它指的是支付一個代價來「買贖」被定罪或被擄的人（出埃及記 21:30；民數記 35:31-32）。耶穌的死，在新約神學中，被理解為：

1. **代替性的**（substitutionary）：他替代了本應受罰的人（以賽亞書 53:5-6）。
2. **獻祭性的**（sacrificial）：他是逾越節的羔羊（哥林多前書 5:7），贖罪日的祭物。
3. **盟約性的**（covenantal）：他的血設立了新約（路加福音 22:20；耶利米書 31:31-34）。
4. **一次完成的**（once-for-all, ephapax）：希伯來書 7:27, 9:12, 10:10 反覆強調這是一次性、不可重複的。

**菩薩道的捨身**在結構上完全不同：

- **沒有「代替性」的概念**：菩薩的犧牲不是代替眾生受刑罰，因為佛教沒有一位要求公義被滿足的位格性審判者。
- **沒有「贖價」的概念**：佛教中的苦來自無明（avidya）和業力（karma），不是來自對一位聖潔之神的得罪。解脫的路徑是覺悟，不是被「贖買」。
- **可重複**：菩薩在無數世中反覆捨身（如本生經 Jataka tales 中的故事），而耶穌的死是一次性、不可重複的歷史事件。
- **動力不同**：耶穌的捨命是出於父神的旨意（約翰福音 10:18：「沒有人奪我的命去，是我自己捨的」），是三一神救贖計畫的執行。菩薩的捨身是出於大悲心，為累積波羅蜜（paramita）。

## 總結：「宗教都是一樣的」這個前提成立嗎？

從嚴謹的文本和歷史研究角度來看，三組比較在**表面行為層次**確實有相似之處（善待他人、肯定人的價值、犧牲自我），但在**形上學基礎**、**神學邏輯**、和**救贖機制**上存在根本性的結構差異：

1. **有神論 vs 非有神論**：聖經的倫理、人論、救贖論都以一位位格性的創造主神為前提；佛教的框架中沒有這樣一位神。
2. **盟約 vs 覺悟**：聖經的核心敘事是盟約（covenant）——神主動發起與人的關係；佛教的核心敘事是覺悟（enlightenment）——從無明中醒來。
3. **恩典 vs 自力**：基督教宣告人無法靠自己達到神的標準，需要神親自介入（以弗所書 2:8-9）；佛教（尤其是南傳）強調自依止、法依止（attadipa, dhammadipa）。
4. **一次性歷史事件 vs 輪迴中的反覆實踐**：耶穌的十字架是在歷史中一次發生的事件，有具體的時間和地點（本丟彼拉多任內）；菩薩道是在無盡的輪迴中反覆修行。

將這些差異說成「都是一樣的」，等於在學術上忽略了兩個傳統各自最核心、最不可協商的宣告。這不是說一個比另一個「好」，而是說它們在回答不同的問題：

- **聖經在問**：聖潔的神如何與有罪的人恢復關係？
- **佛教在問**：有情眾生如何從苦的循環中解脫？

這兩個問題本身就不同，給出的答案自然也不會「一樣」。

### 附注：關於方法論

以上分析純粹是從文本和歷史的角度進行的比較宗教研究，目的是呈現兩個傳統各自的內在邏輯。這不是在評判哪個傳統「對」或「錯」，而是指出：尊重每個傳統，恰恰要求我們認真對待它們之間的差異，而不是用「都一樣」來抹平它們各自的獨特宣告。
