# 以賽亞書 7:14 中的 עַלְמָה (almah) 是否指「處女」？

## 一、經文背景

以賽亞書 7:14（和合本）：

> 因此，主自己要給你們一個兆頭，必有童女懷孕生子，給他起名叫以馬內利。

這節經文是舊約中最具爭議的經文之一，其核心爭議在於希伯來文 **עַלְמָה** (*ʿalmâ*) 一詞究竟應翻譯為「處女」（virgin）還是「年輕女子」（young woman）。

## 二、希伯來文 עַלְמָה 的語義分析

### 1. 詞根與形態

עַלְמָה 來自詞根 **ע-ל-מ** (*ʿ-l-m*)，其基本含義與「隱藏」或「青春期」有關。陽性形式 **עֶלֶם** (*ʿelem*) 意為「年輕男子」（見撒母耳記上 17:56、20:22）。עַלְמָה 是其陰性對應形式，語義核心指向**已達適婚年齡的年輕女子**。

### 2. 舊約中 עַלְמָה 的所有用例

עַלְמָה（含複數形 עֲלָמוֹת）在舊約中僅出現 **9 次**，逐一考察如下：

| 經文 | 上下文 | 是否明確指處女？ |
|------|--------|------------------|
| **創世記 24:43** | 亞伯拉罕的僕人尋找利百加 | 利百加在 24:16 被稱為 בְּתוּלָה（betulah，處女），此處用 עַלְמָה 描述同一人 |
| **出埃及記 2:8** | 摩西的姐姐米利暗 | 年幼女孩，尚未婚嫁 |
| **詩篇 68:25 [68:26]** | 擊鼓的少女們 | 無法判斷是否為處女 |
| **箴言 30:19** | 「男與女交合的道」 | 上下文涉及求愛或性行為，語義曖昧 |
| **雅歌 1:3** | 眾 עֲלָמוֹת 愛慕新郎 | 指年輕女子，未必是處女 |
| **雅歌 6:8** | 與皇后、妃嬪並列的 עֲלָמוֹת | 可能指「未出嫁的少女」，與妃嬪形成對比 |
| **歷代志上 15:20** | 音樂術語 עַל-עֲלָמוֹת | 可能指高音調（女聲音域），非指人 |
| **詩篇 46 標題** | 同上音樂術語 | 同上 |
| **以賽亞書 7:14** | 本節經文 | 正是爭議所在 |

### 3. עַלְמָה 與 בְּתוּלָה 的區別

希伯來文中有一個更明確的詞彙表達「處女」的概念：

- **בְּתוּלָה** (*betulah*)：傳統上被理解為「處女」，在舊約中出現約 50 次。值得注意的是，בְּתוּלָה 在某些經文中也不一定嚴格指「未有性經驗的女子」（例如約珥書 1:8 中，בְּתוּלָה 為已故丈夫哀哭，似乎是已婚的）。

- **עַלְמָה** (*ʿalmah*)：在舊約的用例中，語義重心在於「年輕、適婚年齡的女子」，處女身分可能是其社會文化背景下的附帶含義（古代以色列社會中，未婚的年輕女子通常被預設為處女），但**並非該詞的核心語義成分**。

關鍵問題是：**如果以賽亞要明確表達「處女」的概念，為什麼不使用 בְּתוּלָה？**

## 三、七十士譯本（LXX）的翻譯選擇

公元前三世紀左右，猶太學者將希伯來聖經翻譯為希臘文（即七十士譯本）。在以賽亞書 7:14 中，他們將 עַלְמָה 譯為 **παρθένος** (*parthenos*)，即明確的「處女」。

這個翻譯選擇意義重大：

1. 它表明至少部分希臘化時期的猶太學者認為 עַלְמָה 在此處包含處女的含義。
2. 新約馬太福音 1:23 引用以賽亞書 7:14 時，直接採用了七十士譯本的 παρθένος。
3. 後來的猶太希臘文譯本（阿奎拉 Aquila、辛馬庫斯 Symmachus、狄奧多田 Theodotion）則改譯為 **νεᾶνις** (*neanis*，年輕女子），反映了猶太教與基督教分道揚鑣後的詮釋張力。

## 四、歷史與神學詮釋的分歧

### 基督教傳統立場

基督教傳統（從馬太福音開始）將以賽亞書 7:14 理解為彌賽亞預言，עַלְמָה 被詮釋為「處女」，指向馬利亞的童貞懷孕。理據包括：

- 七十士譯本的翻譯先例
- 神蹟的兆頭（אוֹת）性質：若僅是普通年輕女子懷孕，何以為「兆頭」？
- 舊約中 עַלְמָה 的用例均未指涉已婚或非處女的女子
- 基督論的整體敘事框架

### 猶太教傳統立場

猶太教傳統（特別是中世紀以後的拉比解經）普遍認為：

- עַלְמָה 的語義核心是「年輕女子」而非「處女」
- 經文的直接歷史語境指向亞哈斯王時代的具體事件（可能是以賽亞的妻子或宮中某位女子）
- 以馬內利的出生是當時政治危機的時間標記，而非遙遠未來的預言
- 拉什（Rashi）等重要拉比釋經家均持此觀點

### 現代學術共識

當代希伯來語語言學的主流學術觀點傾向認為：

- עַלְמָה 的核心語義為「適婚年齡的年輕女子」
- 處女身分是文化推論而非詞彙本義
- 烏加里特語（Ugaritic）中的同源詞 *ǵlmt* 也指「年輕女子」，支持此理解
- 以賽亞書 7:14 在原始語境中可能有雙重應驗（近期歷史應驗 + 遠期彌賽亞應驗）

## 五、結論

**עַלְמָה 是否等同於「處女」？嚴格從語言學角度來說，答案是否定的。**

עַלְמָה 的核心語義是「年輕的、適婚年齡的女子」。在古代以色列社會語境中，這樣的年輕女子通常被假定為處女，但「處女」並非 עַלְמָה 一詞的定義性語義成分。如果以賽亞的首要意圖是強調「處女」這一概念，בְּתוּלָה 會是更精確的詞彙選擇。

然而，這並不意味著基督教對此經文的彌賽亞式解讀毫無根據。七十士譯本的翻譯（παρθένος）表明在基督教興起之前，已有猶太學者認為此處蘊含處女的含義。而「兆頭」（אוֹת）一詞的使用也暗示這不僅僅是一個普通的出生事件。

這個問題最終涉及的不只是一個詞彙的語義範圍，而是整個詮釋學框架的選擇：歷史-文法法（historical-grammatical method）傾向「年輕女子」的翻譯，而預表論（typological）或基督論（Christological）解讀則為「處女」的理解提供了神學基礎。
