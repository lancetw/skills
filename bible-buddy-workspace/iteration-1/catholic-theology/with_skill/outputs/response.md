# 路加福音 1:28 —— 「蒙大恩的女子」是否教導天主教的馬利亞教義？

> **釐清方向**：你提到的「無原罪始胎」（Immaculate Conception）、「終身童貞」（Perpetual Virginity）、「天主之母」（Theotokos）、「向馬利亞祈禱代求」（Marian intercession）——這些概念在第一世紀均不存在。它們是後期教會發展出來的教義，不是經文本身的教導。以下逐一分析。

---

## 經文全文：路加福音 1:26-38（天使報喜）

**和合本修訂版 (RCUV, 2010)**（來源：Bible Gateway RCU17TS）：

> [26] 到了第六個月，天使加百列奉 神的差遣往加利利的一座城去，這城名叫拿撒勒，[27] 到一個童女那裏，她已經許配大衛家的一個人，名叫約瑟；童女的名字叫馬利亞。[28] 天使進去，對她說：「蒙大恩的女子，你好，主和你同在！」[29] 馬利亞因這話就很驚慌，又反覆思考這樣問候是甚麼意思。[30] 天使對她說：「馬利亞，不要怕，你在 神面前已經蒙恩了。」[31] 你要懷孕生子，要給他起名叫耶穌。[32] 他將要為大，稱為至高者的兒子；主 神要把他祖先大衛的王位給他。[33] 他要作雅各家的王，直到永遠；他的國沒有窮盡。」[34] 馬利亞對天使說：「我沒有出嫁，怎麼會有這事呢？」[35] 天使回答她說：「聖靈要臨到你身上；至高者的能力要庇蔭你，因此，那要出生的聖者要稱為 神的兒子。[36] 況且，你的親戚伊利莎白，就是那素來稱為不生育的，在年老的時候也懷了男胎，現在懷孕六個月了。[37] 因為，出於 神的話，沒有一句不帶能力的。」[38] 馬利亞說：「我是主的使女，願意照你的話實現在我身上。」於是天使離開她去了。

**呂振中譯本**（來源：信望愛 bible.fhl.net）：

**新漢語譯本 (CCV)**（來源：chinesebible.org.hk）：

> [28] 天使進去，對她說：「蒙大恩的女子，你好！主與你同在！」

**KJV**（來源：信望愛 bible.fhl.net）：

## 希臘文分析：κεχαριτωμένη (kecharitomene)

路加福音是希臘文寫成的，這裏的關鍵詞是 **κεχαριτωμένη** (kecharitomene)。

- **詞根**：χαριτόω (charitoo) —— 意思是「施恩於」、「使蒙恩惠」
- **語態**：被動 (passive) —— 馬利亞是恩典的**接受者**，不是恩典的來源
- **時態**：完成式 (perfect) —— 表示一個已完成、效果延續的動作
- **直譯**：「已被施予恩惠的」、「已蒙恩的」

### 翻譯偏差警示

拉丁武加大譯本（Vulgate）將此詞翻成 **gratia plena**（「滿有恩典」/ "full of grace"），這個翻譯用了主動語態的表述，彷彿恩典是馬利亞內在的特質。但希臘原文是**被動式**——她「被施恩」，是神的主動行為，不是她本身的狀態描述。

中文譯本「蒙大恩的女子」比拉丁文翻譯更忠實於原文——「蒙」字保留了被動含義，表明恩典來自神。

### 同一個字根在別處的使用

以弗所書 1:6 使用了同一字根 χαριτόω：「使我們在愛子裏**蒙恩**」（ἐχαρίτωσεν）。這裏說的是**所有信徒**在基督裏蒙恩。如果 kecharitomene 證明馬利亞無原罪，那以弗所書 1:6 就要證明所有信徒也無原罪——顯然天主教不會這樣主張。

## 逐一檢驗四個天主教教義

### 一、無原罪始胎（Immaculate Conception）

**天主教主張**：「蒙大恩」證明馬利亞從受孕之時就免於原罪。

**第一世紀的事實**：

1. **「原罪」概念本身不存在於第一世紀猶太教。** 「原罪」（Original Sin）是奧古斯丁（Augustine, 354-430 CE）在第四至第五世紀發展出來的教義。第一世紀猶太人的概念是 **יֵצֶר** (yetzer) ——「傾向」，分為善傾向 (yetzer ha-tov) 和惡傾向 (yetzer ha-ra)。這不是遺傳的罪性，而是人人都有的內在掙扎。要「免於」一個尚未被發明的概念，在邏輯上不可能。

2. **「蒙大恩」不等於「無罪」。** kecharitomene 表示「被施予恩惠」，指的是神揀選馬利亞擔任特殊使命。經文自己解釋了這恩惠是什麼——第 30-33 節說明白了：她蒙恩，是因為她將懷孕生下彌賽亞。不是因為她沒有罪。

3. **馬利亞自己稱神為「我的救主」。**

   > 路加福音 1:47：「我靈以 **神我的救主** 為樂」（來源：Bible Gateway RCUV）

   如果馬利亞無罪，她為什麼需要救主？天主教的解釋是「預防性救贖」——神預先赦免她使她免於沾染原罪。但這完全是經文之外的推論，經文本身沒有任何暗示。

4. **保羅明確說「世人都犯了罪」。**

   > 羅馬書 3:23：「因為世人都犯了罪，虧缺了 神的榮耀」（來源：Bible Gateway RCUV）

   πάντες（pantes，「所有人」）沒有附帶例外條款。

5. **教義發展史**：無原罪始胎直到 **1854 年**才由教宗庇護九世正式定為信條（Ineffabilis Deus）。中世紀時，連方濟各會和道明會之間都為此爭論不休。伯爾納（Bernard of Clairvaux, 1090-1153）和阿奎那（Thomas Aquinas, 1225-1274）都反對此教義。

### 二、終身童貞（Perpetual Virginity）

**天主教主張**：馬利亞產前、產時、產後始終是童貞。

**經文怎麼說**：

1. **馬太福音 1:25 明確說**：

   > 「但是沒有和她同房，**直到**她生了兒子，就給他起名叫耶穌。」（來源：Bible Gateway RCUV）

   「直到」（ἕως οὗ / heos hou）的自然語意是：在此之前沒有，在此之後有了。

2. **耶穌有兄弟姊妹——經文多處記載**：

   > 馬太福音 13:55-56：「這不是那木匠的兒子嗎？他母親不是叫馬利亞嗎？他兄弟們不是叫**雅各、約瑟、西門、猶大**嗎？他姊妹們不是都在我們這裏嗎？」（來源：Bible Gateway RCUV）

   > 馬可福音 6:3：「這不是那木匠嗎？不是馬利亞的兒子**雅各、約西、猶大、西門的長兄**嗎？他姊妹們不也是在我們這裏嗎？」（來源：Bible Gateway RCUV）

   > 加拉太書 1:19：「至於別的使徒，除了**主的兄弟雅各**，我都沒有見過。」（來源：Bible Gateway RCUV）

   保羅親自見過雅各，稱他為「主的兄弟」（τὸν ἀδελφὸν τοῦ κυρίου）。

3. **「ἀδελφοί 是堂兄弟」的辯護站不住腳**：希臘文有專門表示「堂兄弟」的字——**ἀνεψιός** (anepsios)，保羅在歌羅西書 4:10 用它稱呼巴拿巴的表弟馬可。如果耶穌的兄弟只是堂兄弟，新約作者為什麼不用 anepsios？他們每一次都用 adelphos（兄弟），而且經文將這些人與「他母親馬利亞」放在一起提到——語境清楚指向同母所生的兄弟。

4. **教義發展史**：終身童貞的文獻記載最早出現在第二世紀末的《雅各原始福音》（Protoevangelium of James），這是一部**偽經**，不被任何正典收錄。正式確認此教義是在 **553 年第二次君士坦丁堡大公會議**，授予馬利亞 Aeiparthenos（永遠童貞）的稱號。

### 三、天主之母（Theotokos / 神之母）

**天主教主張**：馬利亞是「天主之母」（θεοτόκος, Theotokos），因此地位超越一般人。

**第一世紀的事實**：

1. **Theotokos 是基督論用語，不是馬利亞論。** 431 年以弗所大公會議定義 Theotokos，目的是對抗涅斯多留（Nestorius）將基督的神性和人性分離的主張。這是在說耶穌是誰（完全的神又完全的人），不是在說馬利亞是誰。將基督論的爭論變成對馬利亞的崇高定位，是後來的發展。

2. **路加福音 1:28 的天使問候沒有暗示馬利亞有任何超越性地位。** 第一世紀的聽眾會理解這是天使宣告一個使命——就像士師記 6:12 天使對基甸說「大能的勇士啊，耶和華與你同在！」一樣，那是使命的宣告，不是本體論的定義。

3. **耶穌本人兩次公開降低馬利亞母親身份的特殊性**：

   > 馬可福音 3:33-35：「耶穌回答他們：『誰是我的母親？誰是我的兄弟？』就環視那周圍坐着的人，說：『看哪，我的母親，我的兄弟！凡遵行 神旨意的人就是我的兄弟姊妹和母親。』」（來源：Bible Gateway RCUV）

   > 路加福音 11:27-28：「眾人中間有一個女人高聲對他說：『懷你胎乳養你的有福了！』耶穌卻說：『更有福的是聽 神的道而遵守的人！』」（來源：Bible Gateway RCUV）

   注意：路加福音 11:27-28 幾乎就是對「因為馬利亞生了耶穌所以她特別有福」這種想法的直接回應。耶穌將焦點從生物學上的母親身份，轉移到遵行神話語的人身上。

### 四、向馬利亞祈禱代求

**天主教主張**：馬利亞在天上為信徒代求，我們可以向她祈禱。

**第一世紀的事實**：

1. **提摩太前書 2:5 說得非常明確**：

   > 「因為只有一位 神，在 神和人之間也只有一位中保，是成為人的基督耶穌。」（來源：Bible Gateway RCUV）

   εἷς (heis) = **一位**。μεσίτης (mesites) = **中保/中間人**。不是「一位主要中保加上很多次要中保」——是「只有一位」。

2. **第一世紀猶太教不向已死之人祈禱。** 申命記 18:10-12 明確禁止與死者溝通（交鬼、招魂）。雖然天主教辯稱這不是「向死人祈禱」而是「請在天上的聖徒代禱」，但第一世紀猶太教沒有這個區分，也沒有這個實踐。

3. **新約從未記載任何人向馬利亞或任何已逝聖徒祈禱。** 使徒行傳記錄了早期教會的禱告生活——他們向神禱告，奉耶穌的名（使徒行傳 4:24-30），從來沒有向馬利亞或任何人禱告。

4. **路加福音 1:28 完全沒有提到代求。** 天使問候馬利亞、宣告她的使命，和「馬利亞在天上聽人禱告並代求」之間沒有任何邏輯連結。

## 第一世紀的原始理解

回到路加福音 1:28 的第一世紀語境：一個加利利農村的年輕猶太女子，被天使告知神揀選了她承擔一個特殊使命——懷孕生下彌賽亞。天使的問候「蒙大恩的」表明這是神主動的揀選和恩惠，與舊約中神揀選人執行使命的模式一致（亞伯拉罕、摩西、基甸、撒母耳的母親哈拿）。

馬利亞的回應令人敬佩——她謙卑地接受：「我是主的使女，願意照你的話實現在我身上」（1:38）。她的尊歌（Magnificat, 1:46-55）充滿了猶太敬虔的表達，回應了哈拿之歌（撒母耳記上 2:1-10），將一切榮耀歸給神，並稱神為「我的救主」。

第一世紀的猶太聽眾會看到什麼？一位順服的猶太女子被神揀選擔任偉大使命——不多，也不少。

## 常見誤讀總結

| 天主教主張 | 來自路加福音 1:28？ | 實際出現時間 |
|------------|-------------------|-------------|
| 無原罪始胎 | 否。「蒙恩」不等於「無罪」；「原罪」概念本身是 4-5 世紀產物 | 1854 年正式定義 |
| 終身童貞 | 否。經文沒有提到，且多處記載耶穌的兄弟姊妹 | 553 年正式確認 |
| 天主之母 | 否。天使稱她蒙恩，不是定義她的本體地位 | 431 年以弗所大公會議 |
| 向馬利亞祈禱 | 否。經文完全沒有提到代求或禱告 | 3-4 世紀逐漸發展 |
