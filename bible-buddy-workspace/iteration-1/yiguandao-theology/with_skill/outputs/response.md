## 一、約翰福音 10:16 ——「我另外有羊，不屬這圈裏的」

### 經文（來源：Bible Gateway RCUV / 新漢語譯本 / 呂振中譯本）

> **RCUV（和合本修訂版）：**「我另外有羊，不屬這圈裏的，我必須領牠們來，牠們也要聽我的聲音，並且要合成一羣，歸一個牧人。」（約翰福音 10:16）
>
> **新漢語譯本（CCV）：**「我還有別的羊，不屬這個羊圈的，我必須把牠們領來；牠們也要聽我的聲音，並且要合成一羣，同屬一個牧人。」
>
> **呂振中譯本：**「我還有別的羊，不是屬於這一圈的；他們、我也必須領來，他們也必聽我的聲音；這就一羣、一個牧人了！」

完整上下文（約翰福音 10:1-18）：

> [7]「我實實在在地告訴你們，我就是羊的門。[8] 凡在我以前來的都是賊，是強盜；羊沒有聯從他們。[9] 我就是門，凡從我進來的，必得安全，並且可進出，找到草吃。[10] 盜賊來，無非要偷竊、殺害、毀壞；我來了，是要羊得生命，並且得的更豐盛。」
>
> [14]「我是好牧人；我認識我的羊，我的羊也認識我，[15] 正如父認識我，我也認識父一樣；並且我為羊捨命。[16] 我另外有羊，不屬這圈裏的，我必須領牠們來，牠們也要聽我的聲音，並且要合成一羣，歸一個牧人。[17] 為此，我父愛我，因為我把命捨去，好再取回來。[18] 沒有人奪去我的命，是我自己捨的；我有權捨棄，也有權再取回。這是我從我父所受的命令。」

### 一貫道的解讀

一貫道主張：「另外有羊，不屬這圈裏的」證明耶穌承認其他宗教（佛教、道教、伊斯蘭教、儒教）也是真理的管道，因此五教本為一家。

### 第一世紀猶太背景：這段話在說什麼？

這段經文的背景是**以西結書 34 章**——任何第一世紀聽眾都會立刻辨認出來的先知經文。以西結書 34 章是 YHWH 對以色列壞牧人的審判，以及祂要親自來牧養的應許：

> **以西結書 34:11-12（Sefaria 希伯來原文）：**
> כִּי כֹה אָמַר אֲדֹנָי יֱהֹוִה הִנְנִי־אָנִי וְדָרַשְׁתִּי אֶת־צֹאנִי וּבִקַּרְתִּים
> 「因為主 YHWH 如此說：看哪，我——我自己——必搜尋我的羊，巡查牠們。」
>
> **以西結書 34:23（Sefaria 希伯來原文）：**
> וַהֲקִמֹתִי עֲלֵיהֶם רֹעֶה אֶחָד וְרָעָה אֶתְהֶן אֵת עַבְדִּי דָוִיד
> 「我必在他們之上立**一個牧人** (רֹעֶה אֶחָד / ro'eh echad)，就是我的僕人大衛。」
>
> **以西結書 34:24（RCUV）：**「我——耶和華必作他們的神，我的僕人大衛要在他們中間作王。這是我——耶和華說的。」

關鍵觀察：

1. **「圈」（希臘文 αὐλή / aulē）** 指的是字面上的**羊圈**，在第一世紀猶太語境中，「這個圈」最自然的理解是**以色列**（猶太人）。「另外的羊」指的是**散居在外邦中的以色列人**（diaspora），或者是**外邦人中將歸向以色列之神的人**。

2. **以西結書 34 和 37 章的統一主題**：以西結書 37:15-28 明確講到**猶大和以色列（北國）兩根杖合為一根**，由一個牧人（大衛的後裔）統治。「另外有羊」在這個先知傳統中，指的是**被分散的北國以色列支派**重新歸回。

3. **「合成一羣，歸一個牧人」** 希臘文：μία ποίμνη, εἷς ποιμήν（mia poimnē, heis poimēn）。注意：是歸向**一個牧人**——就是耶穌本人。這不是「各有各的牧人」或「殊途同歸」的多元論，而是相反的排他性宣言：所有的羊最終要歸向**同一個**牧人。

4. **約翰福音 10:8 和 10:1 的排他性語言**：同一段講論中，耶穌說「凡在我以前來的都是賊，是強盜」（10:8），「不從門進去的就是賊、就是強盜」（10:1）。如果耶穌真的在 10:16 承認五教都是真理，他為什麼在同一段話中把其他人都稱為「賊」和「強盜」？

5. **第一世紀猶太獨一神論**：

> **申命記 6:4（Sefaria 希伯來原文）：**
> שְׁמַע יִשְׂרָאֵל יְהֹוָה אֱלֹהֵינוּ יְהֹוָה אֶחָד
> *Shema Yisrael YHWH Eloheinu YHWH Echad*
> 「聽啊，以色列！YHWH 是我們的神，YHWH 是唯一的。」
>
> **以賽亞書 43:10-11（Sefaria 希伯來原文）：**
> לְפָנַי לֹא־נוֹצַר אֵל וְאַחֲרַי לֹא יִהְיֶה
> אָנֹכִי אָנֹכִי יְהֹוָה וְאֵין מִבַּלְעָדַי מוֹשִׁיעַ
> 「在我以前沒有任何被造的真神 (אֵל / el)，在我以後也必沒有。我——惟有我——是 YHWH；除我以外沒有拯救者 (מוֹשִׁיעַ / moshia')。」

第一世紀猶太教是激烈的獨一神論。一個加利利猶太拉比絕不可能承認「無極老母」或任何其他神明與 YHWH 平等——這在 Shema 的框架下是不可想像的。

### 結論

一貫道的解讀將「另外有羊」從以西結書 34 章的統一以色列主題中完全抽離，注入了19世紀中國五教合一的宗教融合思想。在第一世紀猶太背景下，這段話不是承認多元宗教，而是宣告：**分散的羊群要被一個牧人（大衛式的彌賽亞）帶回統一的群體中**。

## 二、啟示錄 3:12 ——「新耶路撒冷」≠「理天」

### 經文（來源：Bible Gateway RCUV / 新漢語譯本 / 呂振中譯本）

> **RCUV：**「得勝的，我要使他在我神的殿中作柱子，他必不再從那裏出去。我又要把我神的名和我神城的名——從天上我神那裏降下來的新耶路撒冷，和我的新名，都寫在他上面。」（啟示錄 3:12）
>
> **新漢語譯本（CCV）：**「得勝的，我將使他在我神的殿中作柱子，他也絕不再離開那裏。在他身上我又將要寫上我神的名、我神之城的名——就是從我的神那裏、自天上降下來的新耶路撒冷的名——和我的新名。」
>
> **呂振中譯本：**「得勝的、我必使他做柱石在我的上帝的殿堂中；他決不再出去到外面。我必將我的上帝之名、和我的上帝之城的名（新的耶路撒冷、那由天上、從我的上帝那裏降下來的）、連我新的名、都寫在他上面。」

更完整的新耶路撒冷描述（啟示錄 21:9-14，RCUV）：

> [10] 我在聖靈感動下，天使帶我到一座高大的山，給我看由神那裏、從天而降的聖城耶路撒冷。
> [12] 它有高大的牆，有十二個門，門上有十二位天使，門上又寫着**以色列人十二個支派的名字**。
> [14] 城牆有十二個根基，根基上有**羔羊十二使徒的名字**。

### 一貫道的解讀

一貫道將「新耶路撒冷」等同於一貫道的「理天」——無極老母的居所，靈魂回歸的最終目的地。

### 第一世紀猶太背景：新耶路撒冷的具體含義

1. **新耶路撒冷是具體的「耶路撒冷」**——不是抽象的靈界概念。對第一世紀猶太人而言，「耶路撒冷」(יְרוּשָׁלַיִם / Yerushalayim) 是 YHWH 的聖殿所在地，是大衛的城，是以色列民族的心臟。「新耶路撒冷」是末世更新的耶路撒冷，不是某個非猶太教的天界。

2. **十二支派 + 十二使徒 = 以色列的更新**：啟示錄 21:12-14 明確說城門上刻著**以色列十二支派的名字**，根基上刻著**十二使徒的名字**。這個城市的身份完全是猶太的——它是以色列盟約歷史的高潮，而不是某個超越所有宗教的普世天堂。

3. **啟示錄的文學類型——猶太啟示文學（apocalyptic literature）**：啟示錄屬於猶太啟示文學傳統，與但以理書、以諾書(1 Enoch，約公元前3-1世紀)、以斯拉四書(4 Ezra，約公元100年)同類。這個傳統中的「天上之城」概念來自以西結書 40-48 章的新聖殿異象，不是來自中國民間宗教的「理天」概念。

4. **「理天」概念的時代錯位**：一貫道的「理天、氣天、象天」三界觀是中國民間宗教和道教宇宙論的融合產物，其系統化出現在明清時期（16-19世紀）。這比啟示錄晚了 1500 年以上，也與啟示錄的猶太宇宙觀完全不同。

5. **啟示錄 21:3 的盟約語言**：

> 「看哪，神的帳幕 (σκηνή / skēnē) 在人間！他要和他們**同住**，他們要作**他的子民**。」（RCUV）

希臘文 σκηνή 對應希伯來文 מִשְׁכָּן (mishkan)——會幕。這是出埃及記以來 YHWH 與以色列盟約關係的核心意象：神「住在」(שָׁכַן / shakhan) 祂的百姓當中。新耶路撒冷是**盟約的最終實現**，不是某個超越盟約的宇宙概念。

### 結論

新耶路撒冷從頭到尾都是一個**猶太盟約概念**——以色列的十二支派、十二使徒、YHWH 的聖殿、YHWH 的帳幕在人間。將它等同於一貫道的「理天」，需要完全無視經文中所有具體的猶太元素。

## 三、創世記 2:7 ——「將生命之氣吹進他的鼻孔」≠「點玄關」

### 經文（來源：Sefaria 希伯來原文 / Bible Gateway RCUV / 新漢語譯本 / 呂振中譯本）

> **希伯來原文（Sefaria）：**
> וַיִּיצֶר יְהֹוָה אֱלֹהִים אֶת־הָאָדָם עָפָר מִן־הָאֲדָמָה וַיִּפַּח בְּאַפָּיו נִשְׁמַת חַיִּים וַיְהִי הָאָדָם לְנֶפֶשׁ חַיָּה
>
> **RCUV：**「耶和華神用地上的塵土造人，將生命之氣吹進他的鼻孔，這人就成了有靈的活人。」
>
> **新漢語譯本（CCV）：**「上主神用土地上的塵土塑造人，把生命的氣息吹進他的鼻孔，那人就成了有生命的人。」
>
> **呂振中譯本：**「永恆主上帝用地上的塵土塑造了人，將生氣吹進他的鼻孔裏，那人就成了一個有生命的活人。」

### 一貫道的解讀

一貫道主張：神「將生氣吹在他鼻孔裏」就是「點道」——等同於一貫道點傳師在信徒額頭「玄關」處點的儀式。

### 希伯來原文分析

逐字拆解：

| 希伯來文 | 轉寫 | 意義 |
|----------|------|------|
| וַיִּפַּח | va-yippach | 「他吹」——動詞 נפח (nafach)，「吹氣、呼氣」 |
| בְּאַפָּיו | be-appav | 「在他的鼻孔裏」——אַף (af) = 鼻子/鼻孔，陽性名詞雙數 + 第三人稱後綴 |
| נִשְׁמַת | nishmat | 「氣息（的）」——נְשָׁמָה (neshamah) = 呼吸、氣息 |
| חַיִּים | chayyim | 「生命」——複數形式 |

關鍵字詞：

1. **נְשָׁמָה (neshamah)**：「氣息」。這不是指某個神秘的靈性能量點，而是字面意義上的**呼吸**。當 YHWH 將 neshamah 吹入土人（亞當），土就變成了一個**會呼吸的活物**。

2. **אַף (af)** 的位置：**鼻孔**（鼻子的雙數形式）。不是「額頭」、不是「眉心」——是鼻孔。因為呼吸是通過鼻子進行的。這是古代近東對生命觀察的自然表達：活人呼吸，死人停止呼吸。

3. **נֶפֶשׁ חַיָּה (nefesh chayyah)**：「有生命的存在體」。注意 RCUV 翻譯為「有靈的活人」——這個「靈」字反映了後來的靈魂/身體二元論。希伯來文 נֶפֶשׁ (nefesh) 不是「靈魂」（與肉體對立的非物質實體），而是**整個活生生的存在**——包括血肉、呼吸、渴望、生命力。同一個詞 נֶפֶשׁ חַיָּה 也用在創世記 1:20-21 描述海洋生物和動物。

4. **翻譯偏差提醒**：RCUV 的「有靈的活人」可能暗示「人有靈魂」的柏拉圖式理解。更準確的翻譯是 CCV 的「有生命的人」或呂振中的「有生命的活人」。希伯來原文沒有「靈」這個概念在這裡——只有「氣息」(neshamah) 和「活的存在」(nefesh chayyah)。

### 古代近東平行文獻

創世記 2:7 的意象並非獨一無二——古代近東有大量「神用泥土造人、吹入生命」的平行文獻：

- **阿特拉哈西斯史詩**（Atrahasis Epic，約公元前18世紀，古巴比倫）：神用黏土混合被殺之神的血肉來造人。
- **埃努瑪·埃利什**（Enuma Elish，約公元前12世紀，巴比倫）：瑪爾杜克用金古（Qingu）的血造人。
- **赫利奧波利斯傳統**（埃及）：赫努姆神（Khnum）在陶輪上塑造人的身體。

這些文獻表明：「神塑造人體 + 注入生命」是古代近東共通的創世敘事模式。創世記 2:7 用**以色列獨一神論**的框架重述了這個主題——只有 YHWH 一位造物者，沒有眾神的爭鬥。

### 以西結書 37 章的平行——「吹氣」的先知傳統

以西結書 37:9-10（RCUV）：
> 「氣息啊 (הָרוּחַ / ha-ruach)，要從四方而來，**吹在** (וּפְחִי / u-fechi，同一動詞根 נפח) 這些被殺的人身上，使他們活過來。」

以西結書中的「吹氣使枯骨復活」清楚表明：「吹氣」在希伯來聖經中是**使死物獲得生命的行為**。它的對象是「鼻孔」（呼吸器官），而非「額頭」（非呼吸器官）。它的意義是「賦予生物性的生命」，而非「開啟某個神秘穴位」。

### 「點玄關」概念的來源

一貫道的「玄關」概念來自中國道教內丹術，特別是：
- 「玄關一竅」最早見於宋代以後的道教文獻
- 與印度教/瑜伽的「第三隻眼」(ajna chakra) 概念有平行關係
- 這些概念完全不存在於第一世紀猶太文獻、死海古卷、約瑟夫斯、或任何第二聖殿時期的文獻中

### 結論

創世記 2:7 的「吹氣入鼻孔」是古代近東創世敘事的一部分——YHWH 將呼吸（neshamah）吹入泥人的**鼻孔**（af），使其成為一個**會呼吸的活物**（nefesh chayyah）。位置是鼻孔（呼吸器官），不是額頭（玄關）。概念是生物性的生命賦予，不是靈性穴位的開啟。要將此等同於明清道教內丹術的「點玄關」，需要跨越至少 2500 年的文化鴻溝和完全不同的宇宙觀。

## 綜合評估：「五教合一」框架的根本問題

一貫道「五教合一」的核心主張——儒、釋、道、耶、回都是「無極老母（明明上帝）」差派的——面對第一世紀猶太背景時，有一個根本性的矛盾：

**第一世紀猶太教是激烈的排他性獨一神論。**

> **以賽亞書 43:10（希伯來原文）：**
> לְפָנַי לֹא־נוֹצַר אֵל וְאַחֲרַי לֹא יִהְיֶה
> 「在我以前**沒有任何神被造成** (לֹא־נוֹצַר אֵל)，在我以後**也必沒有** (לֹא יִהְיֶה)。」

YHWH 不是「眾多真理管道之一」——在希伯來聖經的框架中，祂是**唯一的神**，除祂以外沒有拯救者。一個加利利猶太拉比（Yeshua）站在 Shema 的傳統中，不可能認為釋迦牟尼或孔子與他傳講的是同一位神的信息——因為 YHWH 明確宣告「除我以外沒有神」，而佛教甚至不談論造物主神。

一貫道的解經方法是典型的**去脈絡化引用**（proof-texting）：從不同的經卷中抽取隻言片語，剝離其文學、歷史和語言脈絡，然後注入一個外來的宗教框架。這在任何嚴肅的文本分析中——無論是聖經學、古典文獻學、還是法律文本解讀——都是不被接受的方法。

2. **以西結書 37:9-10**：「吹氣」(נפח) 動詞的先知用法，與創世記 2:7 平行。來源：Sefaria.org。
3. **啟示錄 21:9-14**：新耶路撒冷的具體猶太身份（十二支派 + 十二使徒）。來源：Bible Gateway RCUV。
4. **以賽亞書 43:10-12**：YHWH 的排他性獨一神宣告。來源：Sefaria.org 希伯來原文 + Bible Gateway RCUV。
5. **申命記 6:4**（Shema）：以色列信仰的核心宣告。來源：Sefaria.org。
6. **死海古卷**（約公元前2世紀-公元1世紀）：第二聖殿時期猶太文獻中不存在任何「五教合一」或「點玄關」的概念。
7. **阿特拉哈西斯史詩**（Atrahasis，約公元前18世紀）：古代近東「泥土造人」母題的早期文獻。
8. **約瑟夫斯，《猶太古史》**（Antiquities，約93 CE）：第一世紀猶太教的一手見證，記載猶太人對異教的排斥態度。
