# 真耶穌教會五大教義：從第一世紀猶太觀點的審視

## 一、總體評估

真耶穌教會（True Jesus Church, TJC）的五大教義各自引用了真實的經文，但其詮釋方式往往脫離了第一世紀猶太基督徒的歷史脈絡。以下逐一分析。

---

## 二、逐項分析

### 1. 洗禮必須「奉耶穌基督的名」，而非奉三位一體的名

**TJC 引用：** 使徒行傳 2:38（「你們各人要悔改，奉耶穌基督的名受洗」）

**第一世紀猶太觀點：**

- 使徒行傳確實多次記載「奉耶穌的名」施洗（徒 2:38、8:16、10:48、19:5）。這在第一世紀的語境中，「奉某人的名」（eis to onoma）意味著「歸入」或「在……的權柄之下」，強調的是與基督的認同，而非一個精確的洗禮咒語公式。
- 馬太福音 28:19 記載耶穌吩咐「奉父、子、聖靈的名」施洗。早期教會文獻《十二使徒遺訓》（Didache，約主後一世紀末）第 7 章明確指示「奉父、子、聖靈的名」施洗，證明三一名洗禮在極早期已存在。
- 第一世紀猶太基督徒並不認為這兩個公式相互矛盾。「奉耶穌的名」描述的是洗禮的歸屬對象，「奉父、子、聖靈的名」描述的是洗禮的神學內涵。這是描述同一件事的不同面向，而非兩個互斥的儀式規定。
- **面朝下全身浸水**的做法沒有新約經文的直接支持。希臘文 baptizo 確實意指「浸入」，第一世紀猶太人的潔淨禮（mikveh，淨禮池）確實是全身浸入，但姿勢並無特定規定。《十二使徒遺訓》甚至允許在水不足時用澆灌方式施洗（Didache 7:3），顯示早期教會對形式有相當的彈性。

**結論：** 「奉耶穌的名」施洗有經文根據，但將之理解為排他性的口語公式，並否定三一名公式，不符合第一世紀的證據。面朝下的姿勢規定則完全缺乏古代文獻支持。

---

### 2. 說方言是受聖靈的必要憑據

**TJC 引用：** 使徒行傳 2:4（「他們就都被聖靈充滿，按著聖靈所賜的口才說起別國的話來」）

**第一世紀猶太觀點：**

- 五旬節事件確實伴隨說方言（glossolalia），使徒行傳也記載了其他場合的方言現象（徒 10:46、19:6）。
- 然而，使徒行傳中也有多處記載受聖靈但未提及方言的例子（徒 2:41 三千人受洗、8:17 撒瑪利亞人、9:17-18 保羅自己）。如果方言是「必要憑據」，這些遺漏就無法解釋。
- 保羅在哥林多前書 12:30 明確反問：「豈都說方言嗎？」（me pantes glossais lalousin）希臘文語法（me 引導的問句）預期否定回答，意即「不是所有人都說方言」。這直接否定了方言作為普遍性必要憑據的教義。
- 在第一世紀猶太傳統中，聖靈的標記是多元的：預言、智慧、能力、道德果實。將聖靈的憑據縮限為單一現象，不符合猶太人對 ruach ha-kodesh（聖靈）運作的理解。
- 保羅在加拉太書 5:22-23 將聖靈的果子定義為「仁愛、喜樂、和平、忍耐、恩慈、良善、信實、溫柔、節制」，而非任何外在的超自然表現。

**結論：** 方言在初代教會確實存在，但將之提升為受聖靈的唯一必要憑據，既不符合使徒行傳的整體敘事，也直接與保羅的教導相矛盾。

---

### 3. 守安息日（星期六）

**TJC 引用：** 出埃及記 20:8-11（「當記念安息日，守為聖日」）

**第一世紀猶太觀點：**

- 這是最有第一世紀猶太根據的主張。安息日（Shabbat）毫無疑問是猶太信仰的核心。耶穌自己守安息日（路 4:16），早期猶太基督徒也守安息日。
- 然而，需要注意以下幾點：
  - **耶穌重新定義了安息日的意義。** 他宣告「人子是安息日的主」（可 2:28），並且多次在安息日治病，挑戰法利賽人對安息日的嚴格解釋。安息日對耶穌而言是恩典的記號，而非律法主義的規條。
  - **保羅明確教導安息日不是基督徒的束縛。** 歌羅西書 2:16-17：「所以不拘在飲食上，或節期、月朔、安息日，都不可讓人論斷你們。這些原是後事的影兒，那形體卻是基督。」羅馬書 14:5-6 也表達了類似的自由觀。
  - **主日崇拜的轉變並非始於康士坦丁或羅馬教會。** 使徒行傳 20:7 記載門徒在「七日的第一日」（星期天）聚會擘餅；哥林多前書 16:2 保羅指示信徒在「每逢七日的第一日」奉獻；啟示錄 1:10 約翰提到「主日」（kyriake hemera）。教會歷史文獻如《十二使徒遺訓》（Didache 14:1，約主後 90-100 年）和伊格那丟書信（約主後 110 年，致馬格尼西亞人書 9:1）都證實主日崇拜在第一至二世紀已經普遍，遠早於第四世紀的康士坦丁。
  - 第一世紀存在一個過渡期：許多猶太基督徒同時守安息日和主日，外邦基督徒則主要守主日。耶路撒冷會議（徒 15 章）決議不將摩西律法的軛加在外邦信徒身上，安息日也不在要求之列。

**結論：** 安息日在第一世紀猶太基督教中確實被遵守，但說主日崇拜是「羅馬教會竄改」則不符合歷史事實。主日聚會從使徒時代就已開始，且保羅明確教導安息日不再是基督徒的律法義務。

### 4. 洗腳禮是必須遵守的聖禮

**TJC 引用：** 約翰福音 13:14-15（「我是你們的主、你們的夫子，尚且洗你們的腳，你們也當彼此洗腳。我給你們作了榜樣，叫你們照著我向你們所做的去做。」）

**第一世紀猶太觀點：**

- 洗腳在第一世紀巴勒斯坦是常見的待客之禮（路 7:44），通常由僕人或地位較低者執行。耶穌為門徒洗腳的震撼之處在於身分的逆轉——主人服事僕人。
- 耶穌說「你們也當彼此洗腳」，希臘文 hypodeigma（榜樣）一詞的重點是要門徒效法謙卑服事的精神，而非建立一個新的儀式聖禮（sacrament）。
- 新約其餘書信中，洗腳僅在提摩太前書 5:10 出現一次，作為寡婦善行的描述之一，而非教會聖禮的規定。如果洗腳真是與洗禮、聖餐同等重要的聖禮，保羅和其他使徒在書信中的完全沉默就極難解釋。
- 早期教會公認的聖禮（ordinances）只有洗禮和聖餐，這在《十二使徒遺訓》、殉道者游斯丁（Justin Martyr，約主後 150 年）的《第一護教辭》等文獻中都有清楚記載。洗腳禮從未被列入。
- 第一世紀猶太人的思維方式區分 mitzvah（誡命）和 middah（品格榜樣）。耶穌的洗腳更接近 middah——一個關於謙卑的活教材——而非建立新的 mitzvah。

**結論：** 耶穌確實吩咐門徒彼此洗腳，但將之提升為與洗禮、聖餐同等的「聖禮」，在第一世紀教會中沒有先例，也不符合新約整體的見證。

### 5. 聖餐用無酵餅和葡萄汁

**TJC 引用：** 與逾越節的關聯（出 12:8，無酵餅；最後的晚餐）

**第一世紀猶太觀點：**

- **無酵餅（matzah）：** 這一點有相當的猶太根據。最後的晚餐很可能是逾越節晚餐（Seder），根據符類福音的記載（太 26:17、可 14:12、路 22:7-8）。逾越節期間猶太人只吃無酵餅，因此耶穌在最後的晚餐中使用的幾乎確定是無酵餅。保羅在哥林多前書 5:7-8 也用「無酵」作為屬靈純潔的象徵。這個主張有合理的歷史和神學基礎。
- **葡萄汁而非葡萄酒：** 這一點則缺乏第一世紀的根據。耶穌在最後的晚餐中使用的是「葡萄樹的產物」（gennema tes ampelou，可 14:25），在第一世紀的猶太語境中，這個詞指的就是酒（yayin）。逾越節 Seder 的四杯是葡萄酒，而非葡萄汁——這在米示拿（Mishnah, Pesachim 10:1）中有明確記載。未發酵的葡萄汁在古代巴勒斯坦的氣候下難以長期保存（沒有現代巴氏殺菌技術），因此幾乎不可能是常規使用的飲品。使用葡萄汁代替葡萄酒是十九世紀美國禁酒運動的產物（Thomas Bramwell Welch 在 1869 年發明了巴氏殺菌葡萄汁），與第一世紀猶太習俗無關。

**結論：** 使用無酵餅有充分的猶太歷史根據，但堅持使用葡萄汁而非葡萄酒，完全不符合第一世紀猶太實踐，反映的是近代禁酒傳統而非聖經傳統。

## 三、綜合評估

| 教義 | 有經文引用？ | 符合第一世紀猶太觀點？ | 評語 |
|------|:---:|:---:|------|
| 奉耶穌名洗禮 | 是 | 部分 | 有經文根據，但排他性解讀不符合早期教會實踐 |
| 面朝下全身浸水 | 否 | 否 | 浸水有根據，面朝下姿勢無任何古代文獻支持 |
| 方言為聖靈必要憑據 | 是 | 否 | 直接與保羅在哥林多前書的教導矛盾 |
| 守安息日 | 是 | 部分 | 猶太基督徒確實守安息日，但主日崇拜始於使徒時代，非羅馬教會竄改 |
| 洗腳禮為聖禮 | 是 | 否 | 耶穌的吩咐是謙卑的榜樣，早期教會從未將之列為聖禮 |
| 無酵餅 | 是 | 是 | 與逾越節傳統一致，有充分根據 |
| 葡萄汁（非葡萄酒） | 否 | 否 | 反映十九世紀禁酒運動，非第一世紀猶太實踐 |

**總結：** 真耶穌教會的五大教義中，每一項都能找到部分經文支持，但在詮釋上普遍存在以下問題：

1. **選擇性引用（proof-texting）：** 取用支持己方立場的經文，忽略不利的經文（如保羅關於方言和安息日的教導）。
2. **將描述性經文當作規範性經文：** 使徒行傳是歷史敘事，記載了「發生了什麼」，不等於「必須這樣做」。
3. **忽略歷史發展的軌跡：** 第一世紀的教會正處於從猶太教過渡到獨立運動的動態過程中，不能用某一個時間點的快照來建立永恆教義。
4. **近代傳統的回溯投射：** 特別是葡萄汁和面朝下洗禮的規定，明顯是近代產物被回溯投射到聖經時代。

從嚴謹的第一世紀猶太歷史觀點來看，這些主張有部分合理的觀察（如無酵餅、早期奉耶穌名施洗的記載），但整體教義體系的建構方式——將這五項定為「得救必備條件」——在第一世紀的猶太基督教文獻中找不到根據。
