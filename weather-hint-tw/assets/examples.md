# 範例集

## 晚上 — 多雲、空氣差

```
⛅ 台中市  19°C（體感 20°C）
💧 濕度 88%  無風
⛅ 今日 26°~18°  不會下雨
⛅ 明日 24°~18°  不會下雨
😷 空氣不太好
```

忙了一天了吧？辛苦了。
外面涼涼的蠻舒服，今天溫差大，晚上別只穿短袖出去喔。
接下來不會下雨，不過空氣有點霧霧的，出門的話戴個口罩吧。
週五了，明天也不會下雨，週末好好休息。

## 早上 — 下雨、風大

```
🌧️ 新竹  15°C（體感 12°C）
💧 濕度 92%  風大
☂️ 今日 18°~14°  可能下雨
☀️ 明日 22°~16°  不會下雨
```

出門記得帶外套跟傘，空氣倒是還好。
下午雨應該會停，明天放晴，週末可以期待一下。

## ❌ 不要這樣寫

- 「根據氣象資料，目前氣溫 23°C，天氣狀況為局部多雲。」
- 「讓我查一下天氣……」
- 「AQI 109，PM2.5 為 37.8 微克……」
- 機械式列出所有數字
- 用「臺」不用「台」
- 聊天裡重複卡片已經有的數字
- 「還好嗎？」這種太表面的問候
- 每次都用「辛苦了」開頭
- 逐城市各寫一段一模一樣的結構
- 條列式穿搭建議（「建議穿著：短袖 + 外套 + 長褲」）
- 每次都報 7 天預報（只在連假/劇變時才提）
