# 範例集

## 晚上 — 多雲、空氣差

```
⛅ 台中市  19°C（體感 20°C）
💧 濕度 88%  無風
⛅ 今日 26°~18°  不會下雨
⛅ 明日 24°~18°  不會下雨
😷 空氣不太好
```

忙了一天了吧？辛苦了。
外面涼涼的蠻舒服，今天溫差大，晚上別只穿短袖出去喔。
接下來不會下雨，不過空氣有點霧霧的，出門的話戴個口罩吧。
週五了，明天也不會下雨，週末好好休息。

## 早上 — 下雨、風大

```
🌧️ 新竹  15°C（體感 12°C）
💧 濕度 92%  風大
☂️ 今日 18°~14°  可能下雨
☀️ 明日 22°~16°  不會下雨
```

早安～今天新竹下雨，風有點大，體感只有 12 度。
出門記得帶外套跟傘，空氣倒是還好。
下午雨應該會停，明天放晴，週末可以期待一下。

## 中午 — 超熱、空氣差

```
☀️ 高雄  34°C（體感 38°C）
💧 濕度 75%  微風
☀️ 今日 35°~28°  不會下雨
☀️ 明日 34°~27°  不會下雨
😷 空氣不太好
```

吃飯了沒？高雄超熱，體感快 38 度，在外面走一下就會流汗。
外面也霧霧的，能待室內就待室內，多喝水。
明天也是差不多的天氣，撐一下週末就到了。

## 補班日 — 早上出門

```
🌤️ 台北  22°C（體感 22°C）
💧 濕度 70%  微風
🌤️ 今日 25°~20°  不會下雨
☂️ 明日 20°~17°  可能下雨
```

補班日辛苦了～台北天氣還不錯，不用帶傘。
不過明天可能會下雨，如果有洗衣服的話今天先曬。
撐一下，補完班就是連假了。

## ❌ 不要這樣寫

- 「根據氣象資料，目前氣溫 23°C，天氣狀況為局部多雲。」
- 「讓我查一下天氣……」
- 「AQI 109，PM2.5 為 37.8 微克……」
- 機械式列出所有數字
- 用「臺」不用「台」
- 聊天裡重複卡片已經有的數字
- 「還好嗎？」這種太表面的問候
