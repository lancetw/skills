# 資料欄位說明

腳本輸出一行 JSON，分為 `card` 和 `data` 兩個區塊。

## card（卡片資料）

直接組成 code block 輸出，每個欄位一行：

| 欄位 | 說明 | 範例 |
|------|------|------|
| `city` | 天氣 emoji + 城市名 | `⛅ 台中市` |
| `temp` | 溫度 + 體感 | `19°C（體感 20°C）` |
| `detail` | 濕度 + 風況 +（日出日落） | `💧 濕度 88%  無風` |
| `today` | 今日 emoji + 高低溫 + 降雨 | `⛅ 今日 26°~18°  不會下雨` |
| `tomorrow` | 明日 emoji + 高低溫 + 降雨 | `⛅ 明日 24°~18°  不會下雨` |
| `alert`（選用） | 空氣/UV 警示 | `😷 空氣不太好` |

## data（數據摘要）

供撰寫聊天文字用，不要顯示給使用者。

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `city` | 繁中城市名 | 聊天裡帶出 |
| `time` | 台灣時間 (UTC+8) | 判斷時段（見 prompt-guide.md） |
| `temp` / `feel` | 氣溫 / 體感 | 差很多時提（「體感快 30」） |
| `hum` | 濕度 % | >80% 悶、<40% 乾 |
| `wind` | 風速 km/h | ≥30 大風提醒 |
| `uv` | 紫外線 0-11+ | ≥6 提醒防曬 |
| `hourly` | 未來 6 小時降雨 | 判斷等下會不會下雨 |
| `today` | 今日高低溫 + 日落 | 溫差 >10° 提醒保暖 |
| `tomorrow` | 明日高低溫 + 降雨 | 明天天氣變化大時帶一句 |
| `aqi` | AQI 指數 | >100 提醒口罩 |
| `pm25` | PM2.5 μg/m³ | >35 提一下 |
| `days` | 近 3 天假日/補班 | 假日/補班時融入 |
