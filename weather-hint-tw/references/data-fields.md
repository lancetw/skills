# 資料欄位說明

腳本輸出一行 JSON，分為 `display` 和 `data` 兩個區塊。

## display（顯示用）

直接組成 code block 輸出，每個欄位一行：

| 欄位 | 說明 | 範例 |
|------|------|------|
| `地點` | 天氣 emoji + 城市名 | `⛅ 台中市` |
| `溫度` | 溫度 + 體感 | `19°C（體感 20°C）` |
| `天氣` | 濕度 + 風況 +（日出日落） | `💧 濕度 88%  無風` |
| `今日` | emoji + 高低溫 + 降雨機率 | `⛅ 26°~18°  降雨 10%` |
| `明日` | emoji + 高低溫 + 降雨機率 | `☂️ 24°~18°  降雨 70%` |
| `提醒`（選用） | 空氣/UV 警示 | `😷 空氣不太好` |

## data（AI 判斷用）

供撰寫聊天文字用，不要顯示給使用者。

### 現在

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `city` | 繁中城市名 | 聊天裡帶出 |
| `time` | 台灣時間 (UTC+8) | 判斷時段（用 `hints.period` 更快） |
| `temp` / `feel` | 氣溫 / 體感（°C） | 差很多時提（用 `hints.feel_diff` 判斷） |
| `hum` | 濕度 % | >80% 悶、<40% 乾 |
| `wind` | 風速 km/h | ≥30 大風提醒 |
| `uv` | 紫外線 0-11+ | ≥6 提醒防曬 |

### 今日

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `today_high` / `today_low` | 今日最高 / 最低溫（°C） | 用 `hints.temp_range` 判斷溫差 |
| `today_rain` | 今日降雨機率 % | ≥30 帶傘 |
| `sunrise` / `sunset` | 日出 / 日落時間（HH:MM） | 清晨/傍晚時提 |
| `hourly` | 未來 6 小時逐時降雨 | 用 `hints.rain_soon` 更快 |

### 明日

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `tomorrow_high` / `tomorrow_low` | 明日最高 / 最低溫（°C） | 用 `hints.tomorrow_trend` 判斷趨勢 |
| `tomorrow_rain` | 明日降雨機率 % | ≥30 提醒帶傘 |

### 空氣品質

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `aqi` | AQI 指數 | >100 提醒口罩 |
| `pm25` | PM2.5 μg/m³ | >35 提一下 |

### 日期與預報

| 欄位 | 說明 | 活用時機 |
|------|------|---------|
| `days` | 近 3 天假日/上班，格式：`"今天 週日 放假"` / `"明天 週一 上班"` / `"後天 週四 端午節(放假)"` | 星期幾一定融入聊天；放假/上班切換時提醒 |
| `forecast` | 後天~第 7 天每日摘要 | 用 `hints.forecast_change` 判斷是否帶出 |

## hints（預算提示）

程式預先算好的衍生值，**優先用 hints，減少自己算**：

| 欄位 | 型別 | 說明 | 怎麼用 |
|------|------|------|--------|
| `period` | string | 時段：清晨/早上/上午/中午/下午/傍晚/晚上/深夜 | 直接對照 prompt-guide 時段關心表 |
| `temp_range` | number | 今日最高 − 最低 | >10 → 洋蔥穿法提醒 |
| `feel_diff` | number | 體感 − 實際（正=更熱，負=更冷） | 絕對值 ≥3 才提體感 |
| `rain_soon` | bool | 未來 6 小時任一時段降雨 ≥30% | true → 帶傘提醒 |
| `tomorrow_trend` | string | 明顯升溫 / 明顯降溫 / 差不多 | 不是「差不多」就帶一句 |
| `forecast_change` | list\|null | 未來劇變天（有雨/轉涼/變熱） | 有值 → 帶出多天預報；null → 不提 |

## 多城市格式

- 單城市：`{display, data}` 不變
- 多城市：`{"cities": [{display, data}, {display, data}, ...]}`
