# Biblical Fun-Facts Review Report

## Claim 1: 保羅在大馬士革路上從馬背上摔下來（使徒行傳 9:4）

**Verdict: Contains factual error**

Issues found:

1. **Factual error — "馬背上摔下來" (fell from a horse)**: Acts 9:4 says Paul "fell to the ground" (ἔπεσεν ἐπὶ τὴν γῆν), but nowhere in Acts 9, Acts 22, or Acts 26 — the three accounts of the Damascus road event — is a horse mentioned. The idea that Paul was riding a horse comes from later artistic tradition (e.g., Caravaggio's *Conversion on the Way to Damascus*, 1601), not from the biblical text. This is a common anachronism that conflates art history with Scripture.

2. **Verse reference is slightly misleading**: Acts 9:4 specifically records the voice speaking to Paul ("Saul, Saul, why do you persecute me?"). The falling occurs in the same verse, but citing 9:4 to support the horse claim is misleading since the verse contradicts the claim — it says he fell to the ground, with no horse mentioned.

## Claim 2: 約拿在鯨魚肚子裡三天三夜（約拿書 1:17）

**Verdict: Contains oversimplification / anachronism**

Issues found:

1. **Oversimplification — "鯨魚" (whale)**: The Hebrew text of Jonah 1:17 (MT 2:1) uses דָּג גָּדוֹל (dag gadol), meaning "great fish," not "whale." Whales are mammals, not fish. The text deliberately uses a general term. The "whale" tradition comes largely from the KJV rendering of Matthew 12:40 (κῆτος / ketos, "sea monster"), which was translated as "whale." Calling it a whale is a taxonomic anachronism imposed on the text.

2. **Verse reference**: Jonah 1:17 is correct for the Masoretic Text numbering. Note that in some traditions this verse is numbered as Jonah 2:1. The reference is acceptable but could be noted.

## Claim 3: 十誡共有十二條（出埃及記 20 章）

**Verdict: Contains counting error**

Issues found:

1. **Counting error — "十二條" (twelve)**: The Ten Commandments (עֲשֶׂרֶת הַדְּבָרִים, literally "Ten Words") number ten, not twelve. The name itself — 十誡 (Ten Commandments) — contradicts the claim of twelve. Exodus 34:28 and Deuteronomy 4:13, 10:4 all explicitly state the number as ten.

2. **Internal contradiction**: The claim uses the term 十誡 (Ten Commandments) but then says there are twelve. This is self-contradictory.

3. **Verse reference**: Exodus 20 is a correct reference for the Decalogue (though the commandments also appear in Deuteronomy 5). No issue with the reference itself.

## Claim 4: 耶穌在客西馬尼園禱告時流血汗（路加福音 22:44），這證明他只是普通人

**Verdict: Contains theological oversimplification; textual note needed**

Issues found:

1. **Oversimplification / misleading conclusion — "這證明他只是普通人" (this proves he was just an ordinary person)**: Luke 22:44 describes sweat "like drops of blood" (ὡσεὶ θρόμβοι αἵματος), using a simile. The text does not draw the conclusion that this proves Jesus was "just an ordinary person." This is an editorialized interpretation presented as if it were a biblical fact. Mainstream Christian theology (Chalcedonian) holds that the passage shows Jesus's true humanity but does not negate his divinity — this is the doctrine of two natures. Presenting one theological interpretation as a proven conclusion is misleading.

2. **Textual note**: Luke 22:43-44 is absent from some important early manuscripts (e.g., P75, Codex Vaticanus, Codex Alexandrinus first hand). Many modern critical editions bracket these verses or include them with a note. While the verses are well-attested in other manuscript traditions, a responsible fact-check should note the textual uncertainty.

3. **Minor precision issue — "流血汗" (sweated blood)**: The text says his sweat became "like drops of blood" (ὡσεὶ θρόμβοι αἵματος) — this is a simile comparing the appearance or intensity, not necessarily a literal claim of hematidrosis. Stating it as "sweated blood" flattens the comparison.

4. **Verse reference**: Luke 22:44 is correct.
